## 核心となる4つの主張

### 主張1：新しい等式の発見

既存文献に存在しない等式

$$\alpha \cdot \ln n^\ast = R(S^{n^\ast-1}) \cdot \pi$$

を発見・証明した。 $n^\ast = 11$ （§4〜§5で独立導出）を代入すると：

$$\alpha = \frac{90\pi}{\ln 11} = 117.913131\ldots$$

### 主張2：宇宙論的定数の120桁問題の解消

$$\Lambda = 2(n^\ast-2) \cdot \frac{(n^\ast)^{-\alpha}}{l_P^2} = 1.1076 \times 10^{-52}\,\text{m}^{-2}$$

観測値 $1.089 \times 10^{-52}\,\text{m}^{-2}$ に対し誤差1.71%。パラメータ調整なし。

### 主張3：CMB第1ピークの代数的恒等式

$$\ell_1 = D_3(S^9) + D_1(S^9) = \binom{12}{3} = 220$$

観測値220と誤差0.000%。

### 主張4：全音響ピーク式（外部パラメータゼロ）と第一原理検証（v4.6）

$$\ell_k = 220 + (k-1) \times 303$$

$n_{\rm obs}=3$ と $n^\ast=11$ のみから完全に代数的に導出（位相シフト一定近似）。
さらに v4.6 では、理論導出パラメータ群（ $H_0$ 、 $\Omega_\Lambda$ 、 $T_{\rm CMB}$ 、 $\Omega_b h^2$ 、 $w=-1$ ）のみをBoltzmannコードに入力する第一原理経路が全6ピークを誤差0.65%以内で再生することを検証した（[§8.8](docs/05_cmb_peaks.md#88-第一原理力学経路検証と系統誤差の解明v46)）。

## 結論

本理論の核心的主張を7点にまとめる。

第一： 新しい等式 $\alpha \cdot \ln n^\ast = R(S^{n^\ast-1}) \cdot \pi$ を発見・証明した。

第二： 宇宙論的定数の120桁問題をパラメータ調整なしに解消した（誤差1.71%）。

第三： CMB第1音響ピーク位置 $\ell_1 = 220$ が代数的恒等式として誤差0.000%で導かれる。

第四（v2.0）： 全音響ピーク式 $\ell_k = 220 + (k-1) \times 303$ が完全に代数的に導出される。外部パラメータゼロで $\Lambda$ CDM とほぼ同等の精度（平均誤差1.13%）を実現。

第五（v4.0）： CMB温度 $T_{\rm CMB} \approx 2.7285\,{\rm K}$ （誤差0.11%）が外部パラメータ0で導出される。鍵は $n^\ast = 11$ の二重役割：公理Ωから定まる値が $e^+e^-$ 消滅の統計力学と代数的に同一。

第六（v4.4）： 全6要請（B1・B2・O1・O2・M・P）が公理Ωから演繹されることを証明。理論に含まれる外部仮定は Hurwitz 定理とコーシー関数方程式（確立された数学定理）のみ。

第七（v4.6）： 理論導出パラメータ群のみを入力とする第一原理経路（Boltzmannコード直接計算）が全6ピークを誤差0.65%以内で再生することを検証。残差は全て $\Lambda$ の誤差1.71%の伝播（ $\ell_A$ スケール差0.52%）に縮約され、位相構造は完全に再生される。結論は $n_s$ に依存しない（ $k \ge 2$ で感度0.2%未満）。代数経路と第一原理経路の二経路収束により「後付け批判」への定量的応答を確立し、位相シフトの代数的閉形式を反証可能な予想として記録した。

第八（v4.7）： $T_\nu/T_\gamma = (4/n^\ast)^{1/3}$ の全因子をゲージ自由度定理・スピノル次元定理・ $\eta/\zeta$ 統計重み定理の3つの確立された定理により $n_{\rm obs}$ のみの関数として内部化した。熱力学的 $n^\ast_{\rm th} = 2[(n_{\rm obs}-1) + (1-2^{-n_{\rm obs}})\,2^{(n_{\rm obs}+1)/2}]$ と幾何学的 $n^\ast_{\rm geom} = n_{\rm obs} + 2^{n_{\rm obs}}$ の一致は $n_{\rm obs} = 3$ でのみ成立し（定理11.5.1）、 $n^\ast = 11$ の二重役割は偶然の一致ではなく $n_{\rm obs} = 3$ を一意に選び出す整合性定理であることが証明された。

第九（v4.8）： 旧§5.2の観測限界定義に反例 $n = 7$ が存在することを発見し（構成的検証済み）、同値関係を次数付き森田同値に精密化して修復した。次数付けは公理Ωのパリティ $\sigma$ の空間的実現がClifford代数に誘導する $e_i \mapsto -e_i$ そのものであり、外部選択ではない。Wall定理により $n^\ast = \min\{n > 3 \mid n \equiv 3 \pmod 8\} = 11$ が確定し、周期 $8$ の導出から循環が除去された。

第十（v4.9）： 深紫外（プランク長以下）の時空次元が $D = 2$ であることを、公理Ωの $n = 1$ 起源（時空 $1+1$ ）と、観測可能性が $n \ge 3$ を要求すること（定理4.1.1）から証明した。カスケード両端（紫外 $D=2$ ・赤外 $D=4$ ）はCDT・漸近安全重力・スピンフォーム模型のスペクトル次元流と一致し、次元・スケール写像は暗黒エネルギー長さを係数まで再現する。次元流完了スケール $l(3) \approx 1.9 \times 10^{-21}$ m は既存アプローチと13桁異なる反証可能な識別予測である。

単一の公理（公理Ω）から出発し、確立された外部数学定理（Hurwitz・コーシー・Wall）のみを経由して、複数の独立した宇宙論的観測量が記述される。この構造が本理論の本質的な価値である。
