## 3. エネルギー密度関数の唯一性証明

### 3.1 要請

以下の2条件を課す。

要請B1（閉包性）：一次元は自身で完結する。

$$\varepsilon(1) \otimes 1 = \varepsilon(1)$$

要請B2（スケール不変性）：次元の比率のみがエネルギー比率を決める。

$$\frac{\varepsilon(an)}{\varepsilon(n)} = f(a) \quad \text{（$n$ に依存しない）}$$

### 3.2 唯一解の導出

要請B2を満たす連続関数に関するコーシーの関数方程式 $f(ab) = f(a)f(b)$ の解は冪乗則のみである。

指数関数 $e^{-\beta n}$：$f(a) = e^{-\beta(a-1)n}$ となり $n$ に依存するため不適。

対数関数：スケール不変でないため不適。

よってエネルギー密度の唯一解は：

$$\boxed{\varepsilon(n) = E_0 \cdot n^{-\alpha}}$$

ここで $\alpha > 0$ は次元が増すほどエネルギーが薄まることを表す指数、$E_0$ はプランクスケールのエネルギーである。

## 4. 観測次元数 n_obs = 3 の導出

### 4.1 Hurwitz定理による導出

公理Ω（¬∃ と ∃ の共存 = 対立する2状態の共存）の代数的実現には、2つの独立な部分に直和分解できる構造 $A \oplus A$ 型が必要である。

Hurwitz定理（1898）：実数上の除算代数（ノルムを持つ多元体）は次の4つのみ存在する：

| 代数 | 次元 | 可換性 | 結合性 | 空間次元 |
|------|------|--------|--------|----------|
| $\mathbb{R}$ | 1 | ✓ | ✓ | 0 |
| $\mathbb{C}$ | 2 | ✓ | ✓ | 1 |
| $\mathbb{H}$（四元数） | 4 | ✗ | ✓ | 3 |
| $\mathbb{O}$（八元数） | 8 | ✗ | ✗ | 7 |

四元数 $\mathbb{H}$ が観測次元を決める理由：

- $\mathbb{R}$（1次元）：方向がなく観測不可
- $\mathbb{C}$（2次元）：回転＝反転のみ、角運動量なし
- $\mathbb{H}$（4次元）：最初の非可換体、3次元回転が存在 ✓
- $\mathbb{O}$（8次元）：非結合的、物理法則の再現不可 ✗

四元数の虚数部（純虚数四元数）が構成する空間が3次元であり、これが独立した回転（＝観測可能な方向）を持つ最小次元となる。

$$\boxed{n_{\rm obs} = 3}$$

## 6. 基本方程式の完全証明

### 6.1 基本方程式

$$\boxed{\alpha \cdot \ln n^\ast = (n^\ast-1)(n^\ast-2) \cdot \pi}$$

この群のユニタリ表現の基底はMellin変換の基底：

$$\Psi_\alpha(n) = n^{i\alpha} = \exp(i\alpha \ln n)$$

$n = 1$ から $n = n^\ast$ への全位相蓄積：

$$\Phi_L = \alpha \cdot \ln n^\ast \quad \cdots (L)$$

### 6.3 定常波条件：乗法的正準量子化

乗法的座標 $u = \ln n$ での正準量子化：

$$\hat{p} = -i\frac{d}{du} = -in\frac{d}{dn}$$

境界条件：次元の起源（$n=1$, $u=0$）と観測限界（$n=n^\ast$, $u=\ln n^\ast$）で波が消える。

$$\Psi(n=1) = \Psi(n=n^\ast) = 0$$

定常波条件：

$$\alpha \cdot \ln n^\ast = 2\pi k \quad \cdots (*)$$

### 6.4 チャンネル数 k の決定：so(n\*−1) の構造と完全性要請

$S^{n^\ast-1}$ の等長変換群 $\text{SO}(n^\ast-1)$ のLie代数 $\text{so}(n^\ast-1)$ は基底

$$\{L_{ij} \mid 1 \le i < j \le n^\ast-1\}$$

を持ち、その個数は：

$$\dim(\text{so}(n^\ast-1)) = \binom{n^\ast-1}{2} = \frac{(n^\ast-1)(n^\ast-2)}{2}$$

補題（チャンネル位相定理）：

$L_{ij}$ は反対称行列で固有値 $0$（重複度 $n^\ast-3$）、$\pm i$（各1重）。Taylor展開より：

$$\exp(2\pi \cdot L_{ij}) = I$$

各チャンネルが一周するときの位相蓄積 $= 2\pi$。

数値検証：$\text{so}(10)$ の全 $\binom{10}{2} = 45$ 個の基底行列で $\exp(2\pi L_{ij}) = I$ を確認：全45個 True ✓

全チャンネルの全位相：

$$\Phi_R = \binom{n^\ast-1}{2} \times 2\pi = (n^\ast-1)(n^\ast-2) \cdot \pi \quad \cdots (R)$$

要請P（対称性完全性条件）：

定常波のモード数 $k$ は、$S^{n^\ast-1}$ の等長変換群 $\text{SO}(n^\ast-1)$ の全独立チャンネルをちょうど一度実現する値に等しい：

$$k = \dim(\text{so}(n^\ast-1)) = \binom{n^\ast-1}{2}$$

この要請の正当性：
- $k < \binom{n^\ast-1}{2}$ のとき：$S^{n^\ast-1}$ の対称性が部分的にしか励起されず、次元 $n^\ast$ を特徴付ける全ての独立な幾何学的自由度が実現されない。
- $k = \binom{n^\ast-1}{2}$ のとき：全独立チャンネルがちょうど一周する最小かつ完全な実現。
- $k > \binom{n^\ast-1}{2}$ のとき：同一チャンネルの繰り返し（縮退）が生じ、物理的に新しい情報を含まない。

「ちょうど一回の完全な実現」という最小完全性の原理（Ockhamの剃刀の定量的定式化）により、要請Pは自然な選択である。

> 注意：要請Pは公理Ωから純粋に演繹される主張ではなく、系の量子化に際して課す追加の物理的要請である。この要請を明示的に採用することで、基本方程式が確定する。

### 8.4 CMB第1ピークの代数的証明

定理：

$$D_{n_{\rm obs}}(S^{n^\ast-2}) + D_{n_{\rm obs}-2}(S^{n^\ast-2}) = \binom{n^\ast+1}{n_{\rm obs}}$$

証明（純粋代数）：

$n = n^\ast - 1 = 10$ として：

$$D_3(S^9) = \binom{12}{3} - \binom{10}{1} = 220 - 10 = 210$$

$$D_1(S^9) = \binom{10}{1} = 10$$

$$D_3 + D_1
= \left[\binom{12}{3} - \binom{10}{1}\right] + \binom{10}{1}
= \binom{12}{3}
= 220 \qquad \blacksquare$$

$\binom{10}{1}$ 項が完全にキャンセルする。これは $\binom{*}{-1} = 0$（二項係数の基本定理）による純粋な代数的帰結であり、外部パラメータを一切含まない。

なぜ $n_{\rm obs}=3$ でのみ成立するか：

一般形は：

$$D_{n_{\rm obs}} + D_{n_{\rm obs}-2}
= \binom{n^\ast+1}{n_{\rm obs}} - \binom{n^\ast-3}{n_{\rm obs}-4}$$

右辺第2項が消える条件は $n_{\rm obs} - 4 < 0$、すなわち $n_{\rm obs} = 3$（または1）のときのみ。
Hurwitz定理が $n_{\rm obs} = 3$ を一意に決定するとき、このキャンセルが自動的に生じる。

- $n_{\rm obs} = 1$：$D_1 + D_{-1} = 10 \ne 12 = \binom{12}{1}$（不成立）
- $n_{\rm obs} = 3$：$D_3 + D_1 = 220 = \binom{12}{3} = 220$（成立 ✓）
- $n_{\rm obs} = 5$：$D_5 + D_3 = 1992 \ne 792 = \binom{12}{5}$（不成立）

結論：

$$\boxed{\ell_1 = D_3(S^9) + D_1(S^9) = \binom{12}{3} = 220}$$

観測値 220 と誤差 0.000% で一致する。

## 9. 検証可能な予測

本理論から以下の物理量が予測される。各予測の導出状況を明示する。

### 9.1 テンソル・スカラー比

$$\boxed{r = \frac{1}{D_2(S^9)} = \frac{1}{54} \approx 0.0185}$$

導出：乗法的正準量子化された $S^9$ の量子揺らぎに、量子統計の等分配定理を適用する。

$S^9$ 上のモード展開において $\ell = 0, 1$ は非摂動的背景を記述し、摂動の最低次はスカラー（$\ell=2$）である。

- スカラー摂動チャンネル数：$D_2(S^9) = 54$（$\ell=2$ 縮重度）
- テンソル摂動チャンネル数：$1$（$n_{\rm obs}=3$ 次元観測者が等方的に見る spin-2 の $m=0$ シングレット）

等分配定理（量子統計の基本原理）より、独立なモードは等しい揺らぎ振幅 $h_0$ を持つ。$D_2$ 個のスカラーモードは全て観測に加算されるため：

$$P_S \propto D_2 \cdot h_0^2, \quad P_T \propto 1 \cdot h_0^2$$

$$\boxed{r = \frac{P_T}{P_S} = \frac{1}{D_2} = \frac{1}{54} \approx 0.0185}$$

等分配定理の成立は、乗法的正準量子化のハミルトニアン構造から直接導かれる。乗法的座標 $u = \ln n$ での量子化において、$D_2$ 個の各スカラーモードは同一の境界条件・同一の質量・同一のポテンシャルを持つ独立な調和振動子を構成する。量子統計の等分配定理により、同一のハミルトニアンを持つ独立なモードは等しい零点エネルギーを持つ。これは仮定ではなく、量子力学の定理の直接的帰結である。

テンソルシングレット数 $=1$ の証明：$n_{\rm obs}=3$ 次元の等方的観測者が測定する重力波は、$S^9$ 上のスピン2調和テンソルのうち方位角対称成分（$m=0$ シングレット）のみに対応する。$D_2=54$ 個のモードの中でこの条件を満たすのは1個のみである。

以上より $r = 1/D_2$ は等分配定理と等方的観測条件から証明される。

現在の観測上限 $r < 0.036$（BICEP/Keck 2021）の範囲内。CMB-S4実験（2020年代後半）で検証可能。

### 9.2 重力波スペクトル指数

$r = 1/D_2$ が確定した場合、一般相対論的インフレーション理論の consistency relation より：

$$\boxed{n_t = -\frac{r}{8} = -\frac{1}{8 D_2} = -\frac{1}{432} \approx -0.00231}$$

本理論が一般相対論と整合する限り、この関係が成立する。現在未観測。LISA等の将来実験で原理的に検証可能。

> 旧記述について：以前の版で提示した $n_t = -2/\alpha \approx -0.01696$ は、波数 $k$ と次元数 $n$ の対応規則を未確立のまま適用したものであり、根拠が不十分であった。上記の consistency relation による値に訂正する。

### 9.3 暗黒エネルギー状態方程式

$$\boxed{w = -1}$$

導出：本理論では $\varepsilon(n^\ast) = E_0 \cdot (n^\ast)^{-\alpha}$ であり、観測限界次元数 $n^\ast = 11$ は時間的に一定である。宇宙論的定数 $\Lambda \propto \varepsilon(n^\ast)$ は時間変化しないため、

$$\rho_\Lambda = \mathrm{const} \quad \Rightarrow \quad \frac{d\rho_\Lambda}{dt} = 0$$

エネルギー保存則 $\dot{\rho} + 3H(\rho + P) = 0$ に代入すると $w = P/\rho = -1$ が証明される。

これは $\Lambda\text{CDM}$ と完全に整合する証明済みの帰結であり、提案値ではない。

観測値 $w = -1.028 \pm 0.032$（Planck 2018）との差は $|{-1} - ({-1.028})| = 0.028 < \sigma$、すなわち $1\sigma$ 内で整合。$w \neq -1$ への偏差が将来確認される場合は、$n^\ast$ の時間変化を含む拡張理論の課題となる。

### 9.4 有効ニュートリノ数

$$\boxed{N_{\rm eff} \approx 3}$$

導出：$n_{\rm obs} = 3$ から世代数 $N_\nu = 3$ が証明される（§4）。標準模型の電弱理論において世代数が3であることは、ニュートリノの有効数 $N_{\rm eff}$ の整数部分を決定する。

$$N_{\rm eff} = N_\nu + \delta N_{\rm QCD} = 3 + 0.046 = 3.046 \quad \text{（標準模型QCD補正込み）}$$

本理論は $N_\nu = 3$ の整数値を予測し、QCD補正 $0.046$ は標準模型の計算に委ねる。余剰次元 $D_2 = 54$ を通じた熱補正として $N_{\rm eff} = 3(1+1/D_2) = 3.056$ という推定値も構成できるが、観測精度 $\pm 0.17$ のもとでは $3,\;3.046,\;3.056$ はいずれも区別不可能であり、整数予測 $N_\nu = 3$ が本理論の証明済み帰結である。

観測値 $N_{\rm eff} = 2.99 \pm 0.17$（Planck 2018）との差は $|3 - 2.99|/0.17 = 0.06\sigma$、すなわち $1\sigma$ 内で整合。

### 9.5 バリオン-光子比と脱結合赤方偏移

$$\boxed{R_{\rm dec} = \frac{3(n^\ast - n_{\rm obs})}{4(n^\ast - 1)} = \frac{3 \times 8}{4 \times 10} = \frac{3}{5} = 0.600}$$

導出：$R_{\rm dec} \equiv 3\rho_b/(4\rho_\gamma)$ をバリオン・光子エネルギー密度の縮重度から求める。

$S^{n^\ast-1}$ 上のモードを粒子種に同定する：

- 光子モード：spin-1球面調和の縮重度 $D_1(S^9) = n^\ast - 1 = 10$
- バリオン的モード：spin-2から spin-1を除いた縮重度 $D_2 - D_1 = n^\ast(n^\ast-3)/2 = 44$
- 全対称モード（光子の全自由度）：$\binom{n^\ast}{2} = n^\ast(n^\ast-1)/2 = 55$

密度比は：

$$\frac{\rho_b}{\rho_\gamma} = \frac{D_2 - D_1}{\binom{n^\ast}{2}} = \frac{n^\ast(n^\ast-3)/2}{n^\ast(n^\ast-1)/2} = \frac{n^\ast - 3}{n^\ast - 1} = \frac{8}{10}$$

$$R_{\rm dec} = \frac{3}{4} \cdot \frac{\rho_b}{\rho_\gamma} = \frac{3(n^\ast - n_{\rm obs})}{4(n^\ast - 1)} = \frac{3}{5}$$

観測値 $R_{\rm dec} = 0.603$（Planck 2018）との誤差：0.5%。

脱結合赤方偏移 $z_{\rm dec}$ も代数的に導出される。$\ell_1 = C(n^\ast+1, n_{\rm obs}) = 220$ と $n^\ast$ を用いて：

$$\boxed{z_{\rm dec} = \frac{n^\ast - 1}{2} \cdot \binom{n^\ast+1}{n_{\rm obs}} - n^\ast = 5 \times 220 - 11 = 1089}$$

観測値 $z_{\rm dec} = 1089.80$（Planck 2018）との誤差：0.07%。

### 9.6 ハッブル定数・宇宙論的密度パラメータ・バリオン密度

宇宙論的定数 $\Lambda$（§7で導出済み）から、さらに以下が連鎖的に導出される。

$$\rho_\Lambda = \frac{\Lambda c^2}{8\pi G}$$

$$\boxed{H_0 = \sqrt{\frac{8\pi G\,\rho_\Lambda}{3\,\Omega_\Lambda}} \approx 67.94 \text{ km/s/Mpc}}$$

ここで $\Omega_\Lambda = \rho_\Lambda/\rho_{\rm crit}$ は自己無撞着に決まる。理論値と観測値の比較：

| 量 | 理論値 | 観測値（Planck 2018） | 誤差 |

| $H_0$ | $67.94$ km/s/Mpc | $67.66$ km/s/Mpc | 0.41% |
| $\Omega_\Lambda$ | $0.690$ | $0.6847$ | 0.77% |

バリオン密度は $R_{\rm dec}$（§9.5）と $z_{\rm dec}$（§9.5）および光子密度パラメータ $\Omega_\gamma$（CMB温度 $T_{\rm CMB}$ に依存）から：

$$\Omega_b h^2 = \frac{4}{3} \cdot R_{\rm dec} \cdot (1 + z_{\rm dec}) \cdot \Omega_\gamma h^2$$

$R_{\rm dec} = 3/5$、$1+z_{\rm dec} = 1090$、$\Omega_\gamma h^2 = 2.473 \times 10^{-5}$（観測値）を代入すると：

$$\Omega_b h^2 = \frac{4}{3} \times \frac{3}{5} \times 1090 \times 2.473 \times 10^{-5} = 0.02156$$

## 付録D：全ピーク式 v2.0 の代数的証明の詳細

定理：$n_{\rm obs}=3$, $n^\ast=11$ のとき

$$\ell_k = 220 + (k-1) \times 303 \quad (k = 1, 2, 3, \ldots)$$

証明：

ステップ1：$\ell_1 = 220$ は§8.4で代数的に証明済みである。

ステップ2：音響スケール $\Delta\ell = 303$ の計算。

$$D_4(S^9) = \binom{13}{4} - \binom{11}{2} = 715 - 55 = 660$$

$$D_2(S^9) = \binom{11}{2} - \binom{9}{0} = 55 - 1 = 54$$

$$\Delta\ell = \frac{D_{n_{\rm obs}+1} - D_{n_{\rm obs}-1}}{2} = \frac{660 - 54}{2} = 303$$

ステップ3：$\Delta\ell$ の音響スケールとしての同定。

$S^9$ 上の偶パリティモードは静的背景場を記述し、そのモード密度の変化 $D_4 - D_2 = 606$ が「次の音響振動が起こるための空間的容量の増分」を表す。音響定在波はその進行波2方向の重ね合わせであるから、実効的な $\ell$ 空間間隔は $606/2 = 303$ となる。

ステップ4：全ピーク式の成立。

第1ピーク $\ell_1 = 220$ から始まり、間隔 $\Delta\ell = 303$ で等差数列をなす：

$$\ell_k = \ell_1 + (k-1)\Delta\ell = 220 + (k-1) \times 303 \qquad \blacksquare$$
