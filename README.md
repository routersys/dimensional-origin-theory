## 3. エネルギー密度関数の唯一性証明

### 3.1 要請

以下の2条件を課す。

要請B1（閉包性）：一次元は自身で完結する。

$$\varepsilon(1) \otimes 1 = \varepsilon(1)$$

要請B2（スケール不変性）：次元の比率のみがエネルギー比率を決める。

$$\frac{\varepsilon(an)}{\varepsilon(n)} = f(a) \quad \text{（$n$ に依存しない）}$$

### 3.2 唯一解の導出

要請B2を満たす連続関数に関するコーシーの関数方程式 $f(ab) = f(a)f(b)$ の解は冪乗則のみである。

指数関数 $e^{-\beta n}$：$f(a) = e^{-\beta(a-1)n}$ となり $n$ に依存するため不適。

対数関数：スケール不変でないため不適。

よってエネルギー密度の唯一解は：

$$\boxed{\varepsilon(n) = E_0 \cdot n^{-\alpha}}$$

ここで $\alpha > 0$ は次元が増すほどエネルギーが薄まることを表す指数、$E_0$ はプランクスケールのエネルギーである。

## 4. 観測次元数 n_obs = 3 の導出

### 4.1 Hurwitz定理による導出

公理Ω（¬∃ と ∃ の共存 = 対立する2状態の共存）の代数的実現には、2つの独立な部分に直和分解できる構造 $A \oplus A$ 型が必要である。

Hurwitz定理（1898）：実数上の除算代数（ノルムを持つ多元体）は次の4つのみ存在する：

| 代数 | 次元 | 可換性 | 結合性 | 空間次元 |
|------|------|--------|--------|----------|
| $\mathbb{R}$ | 1 | ✓ | ✓ | 0 |
| $\mathbb{C}$ | 2 | ✓ | ✓ | 1 |
| $\mathbb{H}$（四元数） | 4 | ✗ | ✓ | 3 |
| $\mathbb{O}$（八元数） | 8 | ✗ | ✗ | 7 |

四元数 $\mathbb{H}$ が観測次元を決める理由：

- $\mathbb{R}$（1次元）：方向がなく観測不可
- $\mathbb{C}$（2次元）：回転＝反転のみ、角運動量なし
- $\mathbb{H}$（4次元）：最初の非可換体、3次元回転が存在 ✓
- $\mathbb{O}$（8次元）：非結合的、物理法則の再現不可 ✗

四元数の虚数部（純虚数四元数）が構成する空間が3次元であり、これが独立した回転（＝観測可能な方向）を持つ最小次元となる。

$$\boxed{n_{\rm obs} = 3}$$

## 6. 基本方程式の完全証明

### 6.1 基本方程式

$$\boxed{\alpha \cdot \ln n^\ast = (n^\ast-1)(n^\ast-2) \cdot \pi}$$

$n^\ast = 11$ を代入すると：

$$\alpha = \frac{90\pi}{\ln 11} = 117.913131\ldots$$

### 6.2 左辺の起源：乗法的群の表現論

$\varepsilon(n) = E_0 n^{-\alpha}$ のスケール不変性より、次元 $n$ は乗法的群 $(\mathbb{R}_+, \times)$ の構造を持つ。
この群のユニタリ表現の基底はMellin変換の基底：

$$\Psi_\alpha(n) = n^{i\alpha} = \exp(i\alpha \ln n)$$

$n = 1$ から $n = n^\ast$ への全位相蓄積：

$$\Phi_L = \alpha \cdot \ln n^\ast \quad \cdots (L)$$

### 6.3 定常波条件：乗法的正準量子化

乗法的座標 $u = \ln n$ での正準量子化：

$$\hat{p} = -i\frac{d}{du} = -in\frac{d}{dn}$$

境界条件：次元の起源（$n=1$, $u=0$）と観測限界（$n=n^\ast$, $u=\ln n^\ast$）で波が消える。

$$\Psi(n=1) = \Psi(n=n^\ast) = 0$$

定常波条件：

$$\alpha \cdot \ln n^\ast = 2\pi k \quad \cdots (*)$$

### 6.4 チャンネル数 k の決定：so(n\*−1) の構造

$S^{n^\ast-1}$ の等長変換群 $\text{SO}(n^\ast-1)$ のLie代数 $\text{so}(n^\ast-1)$ は基底

$$\{L_{ij} \mid 1 \le i < j \le n^\ast-1\}$$

を持ち、その個数は：

$$\dim(\text{so}(n^\ast-1)) = \binom{n^\ast-1}{2} = \frac{(n^\ast-1)(n^\ast-2)}{2}$$

補題（チャンネル位相定理）：

$L_{ij}$ は反対称行列で固有値 $0$（重複度 $n^\ast-3$）、$\pm i$（各1重）。Taylor展開より：

$$\exp(2\pi \cdot L_{ij}) = I$$

各チャンネルが一周するときの位相蓄積 $= 2\pi$。

数値検証：$\text{so}(10)$ の全 $\binom{10}{2} = 45$ 個の基底行列で $\exp(2\pi L_{ij}) = I$ を確認：全45個 True ✓

全チャンネルの全位相：

$$\Phi_R = \binom{n^\ast-1}{2} \times 2\pi = (n^\ast-1)(n^\ast-2) \cdot \pi \quad \cdots (R)$$

### 6.5 基本方程式の完成

$(*)$ のモード数 $k$ が $S^{n^\ast-1}$ の全干渉チャンネル数に等しいとき：

$$k = \binom{n^\ast-1}{2} = \frac{(n^\ast-1)(n^\ast-2)}{2} = 45 \quad (n^\ast=11 \text{ のとき})$$

$(L) = (R)$ より：

$$\alpha \cdot \ln n^\ast = (n^\ast-1)(n^\ast-2) \cdot \pi \qquad \blacksquare$$

### 6.6 幾何学的再解釈

基本方程式は次のように書き換えられる：

$$\alpha \cdot \ln n^\ast = R(S^{n^\ast-1}) \cdot \pi$$

左辺は乗法的群のMellin表現の全位相、右辺は $S^{n^\ast-1}$ の球面スカラー曲率

## 8. CMB温度異方性ピーク構造の導出

### 8.1 空間球面の決定

### 8.3 奇パリティ論証によるモード選択

公理Ω（$\neg\exists\to\exists$）は奇パリティ変換である。$S^9$ 上の球面調和のパリティは：

$$Y_\ell(-\Omega) = (-1)^\ell Y_\ell(\Omega)$$

- $\ell$ = 偶数：偶パリティ（対称）→ 静的・均一な背景場 → 観測不可能な背景
- $\ell$ = 奇数：奇パリティ（反対称）→ 動的・音響振動モード → 観測可能

3次元観測者が観測可能な範囲（$\ell \le n_{\rm obs} = 3$）の奇パリティモード：

$$\ell \in \{1, 3\}$$

### 8.4 CMB第1ピークの代数的証明

定理：

$$D_{n_{\rm obs}}(S^{n^\ast-2}) + D_{n_{\rm obs}-2}(S^{n^\ast-2}) = \binom{n^\ast+1}{n_{\rm obs}}$$

証明（純粋代数）：

$n = n^\ast - 1 = 10$ として：

$$D_3(S^9) = \binom{12}{3} - \binom{10}{1} = 220 - 10 = 210$$

$$D_1(S^9) = \binom{10}{1} = 10$$

$$D_3 + D_1
= \left[\binom{12}{3} - \binom{10}{1}\right] + \binom{10}{1}
= \binom{12}{3}
= 220 \qquad \blacksquare$$

$\binom{10}{1}$ 項が完全にキャンセルする。これは $\binom{*}{-1} = 0$（二項係数の基本定理）による純粋な代数的帰結であり、外部パラメータを一切含まない。

なぜ $n_{\rm obs}=3$ でのみ成立するか：

一般形は：

$$D_{n_{\rm obs}} + D_{n_{\rm obs}-2}
= \binom{n^\ast+1}{n_{\rm obs}} - \binom{n^\ast-3}{n_{\rm obs}-4}$$

右辺第2項が消える条件は $n_{\rm obs} - 4 < 0$、すなわち $n_{\rm obs} = 3$（または1）のときのみ。
Hurwitz定理が $n_{\rm obs} = 3$ を一意に決定するとき、このキャンセルが自動的に生じる。

数値的確認：

- $n_{\rm obs} = 1$：$D_1 + D_{-1} = 10 \ne 12 = \binom{12}{1}$（不成立）
- $n_{\rm obs} = 3$：$D_3 + D_1 = 220 = \binom{12}{3} = 220$（成立 ✓）
- $n_{\rm obs} = 5$：$D_5 + D_3 = 1992 \ne 792 = \binom{12}{5}$（不成立）

結論：

$$\boxed{\ell_1 = D_3(S^9) + D_1(S^9) = \binom{12}{3} = 220}$$

観測値 220 と誤差 0.000% で一致する。

### 8.5 組み合わせ論的意味

$\binom{n^\ast+1}{n_{\rm obs}} = \binom{12}{3} = 220$ は
「$n^\ast+1 = 12$ 次元時空の中で $n_{\rm obs} = 3$ 次元断面を選ぶ独立な配置数」を表す。
これが一次元起源の空間が3次元観測者に「見える」最大モード数＝第1ピーク位置となる。
