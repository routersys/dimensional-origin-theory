## 核心となる4つの主張

### 主張1：新しい等式の発見

既存文献に存在しない等式

$$\alpha \cdot \ln n^\ast = R(S^{n^\ast-1}) \cdot \pi$$

を発見・証明した。$n^\ast = 11$（§4〜§5で独立導出）を代入すると：

$$\alpha = \frac{90\pi}{\ln 11} = 117.913131\ldots$$

### 主張2：宇宙論的定数の120桁問題の解消

$$\Lambda = 2(n^\ast-2) \cdot \frac{(n^\ast)^{-\alpha}}{l_P^2} = 1.1076 \times 10^{-52}\,\text{m}^{-2}$$

観測値 $1.089 \times 10^{-52}\,\text{m}^{-2}$ に対し誤差1.71%。パラメータ調整なし。

### 主張3：CMB第1ピークの代数的恒等式

$$\ell_1 = D_3(S^9) + D_1(S^9) = \binom{12}{3} = 220$$

観測値220と誤差0.000%。

### 主張4：全音響ピーク式（外部パラメータゼロ）と力学経路検証（v4.5）

$$\ell_k = 220 + (k-1) \times 303$$

$n_{\rm obs}=3$ と $n^\ast=11$ のみから完全に代数的に導出（位相シフト一定近似）。
さらに v4.5 では、理論導出パラメータ群（$H_0$、$\Omega_\Lambda$、$T_{\rm CMB}$、$\Omega_b h^2$、$z_{\rm dec}$、$R_{\rm dec}$、$w=-1$）のみを標準音響物理に入力する独立な力学経路が第1〜3ピークを誤差0.3%で再生することを検証した（[§8.8](docs/05_cmb_peaks.md#88-力学経路検証と系統誤差の解明v45)）。

## 結論

本理論の核心的主張を7点にまとめる。

第一： 新しい等式 $\alpha \cdot \ln n^\ast = R(S^{n^\ast-1}) \cdot \pi$ を発見・証明した。

第二： 宇宙論的定数の120桁問題をパラメータ調整なしに解消した（誤差1.71%）。

第三： CMB第1音響ピーク位置 $\ell_1 = 220$ が代数的恒等式として誤差0.000%で導かれる。

第四（v2.0）： 全音響ピーク式 $\ell_k = 220 + (k-1) \times 303$ が完全に代数的に導出される。外部パラメータゼロで $\Lambda$CDM とほぼ同等の精度（平均誤差1.13%）を実現。

第五（v4.0）： CMB温度 $T_{\rm CMB} \approx 2.7285\,{\rm K}$（誤差0.11%）が外部パラメータ0で導出される。鍵は $n^\ast = 11$ の二重役割：公理Ωから定まる値が $e^+e^-$ 消滅の統計力学と代数的に同一。

第六（v4.4）： 全6要請（B1・B2・O1・O2・M・P）が公理Ωから演繹されることを証明。理論に含まれる外部仮定は Hurwitz 定理とコーシー関数方程式（確立された数学定理）のみ。

第七（v4.5）： 理論導出パラメータ群のみを入力とする力学経路（標準音響物理の積分計算）が第1〜3ピークを誤差0.3%で再生することを検証。代数式の $k \ge 2$ 系統誤差の正体が $k$ 依存位相シフト $\delta\varphi_k$ の無視であることを解明し、代数経路と力学経路の二経路収束により「後付け批判」への定量的応答を確立した。

単一の公理（公理Ω）から出発し、確立された外部数学定理のみを経由して、複数の独立した宇宙論的観測量が記述される。この構造が本理論の本質的な価値である。
