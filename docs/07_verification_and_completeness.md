### 11.4 残差誤差の物理的解釈

代数式 $\ell_k = 220 + (k-1)\times 303$ は位相シフト一定近似（ $\varphi_k = \bar\varphi$ ）における式である。 $k \ge 2$ の誤差の正体は $k$ 依存の相対位相シフト $\delta\varphi_k$ （輻射駆動効果とDoppler効果に起因、標準音響物理）の無視であり、これを自動的に含む第一原理経路（§8.8）は全6ピークを誤差0.47〜0.65%で再生する。この残差は全て $\ell_A$ のスケール差0.52%（ $\Lambda$ の誤差1.71%の伝播）に縮約され、位相構造は完全に再生されている。

### 11.5 CMB温度 $T_{\rm CMB}$ の導出

$T_{\rm CMB}$ は本理論から代数的に導出可能である。

#### 導出の核心： $n^\ast = 11$ の二重役割

$n^\ast = 11$ は二つの独立な文脈で同じ値として現れる。

役割1（スケール）： 再加熱エネルギー密度

$$\rho_{\rm inf} = \rho_P \cdot (n^\ast)^{-\alpha} \cdot \frac{2}{n_{\rm obs}\,\pi}$$

役割2（温度比）： ニュートリノ-光子温度分離

$$\frac{T_\nu}{T_\gamma} = \left(\frac{4}{n^\ast}\right)^{1/3} = \left(\frac{4}{11}\right)^{1/3}$$

役割2の代数的起源は次のとおりである。 $e^+e^-$ 消滅前後のエントロピー保存において

$$g^{\ast}_{s}(\gamma + e^+e^-) = 2 + \frac{7}{8}\times 4 = \frac{4+7}{2} = \frac{n^\ast}{2}$$

という分解が現れる。分子 $n^\ast = 11 = 4 + 7$ は光子の統計的自由度（ $2 \times 2 = 4$ ）とフェルミオン統計重み（ $7$ ）の和として代数的に決まり、本理論が公理Ωから独立に導出した $n^\ast = 11$ と一致する。

#### 定理11.5.1（熱力学的 $n^\ast$ の内部導出と $n_{\rm obs} = 3$ の一意性）

各因子（ $2$ ・ $7/8$ ・ $4$ ）を $n_{\rm obs}$ のみの関数として内部化し、幾何学的 $n^\ast$ との一致が $n_{\rm obs} = 3$ でのみ成立することを証明する。

**準備（3つの確立された数学・物理定理）：**

(i) ゲージ自由度定理： $d = n_{\rm obs}+1$ 次元時空の質量ゼロベクトル場の物理自由度は $d - 2 = n_{\rm obs} - 1$ 。

(ii) スピノル次元定理： $d$ 次元時空（ $d$ 偶数）のDirac場のオンシェル自由度（粒子・反粒子×ヘリシティ）は $2^{d/2} = 2^{(n_{\rm obs}+1)/2}$ 。これはClifford代数 $\mathrm{Cl}(1, n_{\rm obs})$ の既約表現次元から従う。

(iii) 統計重み定理： $n_{\rm obs}$ 空間次元の質量ゼロ気体のフェルミ・ボーズ間のエネルギー密度比およびエントロピー密度比は、ディリクレ・イータとゼータの比

$$\frac{\eta(n_{\rm obs}+1)}{\zeta(n_{\rm obs}+1)} = 1 - 2^{-n_{\rm obs}}$$

である（ $\eta(s) = (1-2^{1-s})\zeta(s)$ ）。 $n_{\rm obs} = 3$ で $7/8$ 。

**定義：** $e^+e^-$ 消滅直前のエントロピー的有効自由度を内部量のみで

$$g^\ast_s = \underbrace{(n_{\rm obs}-1)}_{\text{(i)}} + \underbrace{\left(1 - 2^{-n_{\rm obs}}\right)}_{\text{(iii)}} \cdot \underbrace{2^{(n_{\rm obs}+1)/2}}_{\text{(ii)}}$$

と定義し、熱力学的観測限界次元数を $n^\ast_{\rm th} := 2\,g^\ast_s$ 、幾何学的観測限界次元数を $n^\ast_{\rm geom} := n_{\rm obs} + 2^{n_{\rm obs}}$ （§5）とする。

**定理：** $n^\ast_{\rm th} = n^\ast_{\rm geom}$ が成立する必要十分条件は $n_{\rm obs} = 3$ であり、そのとき共通値は $11$ である。

**証明：** $n_{\rm obs} = 3$ で $n^\ast_{\rm th} = 2[2 + \tfrac{7}{8}\cdot 4] = 11 = 3 + 8 = n^\ast_{\rm geom}$ （十分性）。一意性： $n \ge 4$ では

$$n^\ast_{\rm geom} - n^\ast_{\rm th} = 2^n - 2^{\lfloor(n+1)/2\rfloor+1}\left(1-2^{-n}\right) + n - 2(n-1)$$

であり、 $n \ge 4$ のとき $n - \lfloor(n+1)/2\rfloor - 1 \ge 1$ から $2^n \ge 2 \cdot 2^{\lfloor(n+1)/2\rfloor+1}$ となるため差は正、かつ $n$ について単調増大する（数値確認： $n=4$ で $6.5$ 、 $n=15$ で $32243$ ）。 $n = 1, 2$ は直接計算で不一致（ $2 \ne 3$ 、 $5 \ne 6$ ）。よって一致は $n_{\rm obs} = 3$ のみ。 $\square$

**系11.5.2：** スピノル自由度が四元数の実次元と一致する条件 $2^{(n_{\rm obs}+1)/2} = \dim_{\mathbb{R}}\mathbb{H} = 4$ もまた $n_{\rm obs} = 3$ でのみ成立する。すなわち $T_\nu/T_\gamma = (4/n^\ast)^{1/3}$ の分子 $4$ は、 $\mathrm{Spin}(3) \cong Sp(1) \subset \mathbb{H}$ （単位四元数群）のスピノル表現空間が $\mathbb{H}$ 自身であることの帰結として、Cayley-Dickson 塔の第2段代数 $\mathbb{H}$ に内部化される。

**帰結（借用問題の解消）：** $T_\nu/T_\gamma = (4/n^\ast)^{1/3}$ に現れる全ての数は標準模型の粒子表からの借用ではなく、(i)〜(iii) の確立された定理を経由した $n_{\rm obs}$ の関数である。さらに熱力学経路と幾何学経路（§5）が同一の $n^\ast = 11$ を与えることは仮定ではなく、 $n_{\rm obs} = 3$ を一意に選び出す整合性定理である。

#### 定理11.5.3（消滅種数 $k = 1$ の一意性）

定理： 消滅する最小Dirac種の個数を $k$ とし、

$$n^\ast_{\rm th}(n_{\rm obs}, k) = 2\left[(n_{\rm obs}-1) + \left(1-2^{-n_{\rm obs}}\right) k\, 2^{(n_{\rm obs}+1)/2}\right]$$

と一般化する。 $n^\ast_{\rm th}(n_{\rm obs}, k) = n^\ast_{\rm geom}(n_{\rm obs}) = n_{\rm obs} + 2^{n_{\rm obs}}$ を満たす整数解は $(n_{\rm obs}, k) = (3, 1)$ のみである。

**証明：** $n_{\rm obs} = 3$ では条件は $4 + 7k = 11$ となり $k = 1$ が唯一の解（ $k=2$ は $18$ 、 $k=3$ は $25$ ）。格子 $1 \le n_{\rm obs} \le 15$ 、 $1 \le k \le 10$ の全数走査で解は $(3,1)$ のみであることを厳密分数計算で確認した。 $n_{\rm obs} \ge 4$ では $2^{n_{\rm obs}}$ の成長がフェルミオン項の成長 $k \cdot 2^{(n_{\rm obs}+1)/2}$ を任意の固定 $k$ で支配するため、大きい $n_{\rm obs}$ にも解は存在しない。 $\square$

**物理的必然性：** もし2種が消滅すれば $T_\nu/T_\gamma = (4/18)^{1/3}$ となり、誤差0.11%で成功している $T_{\rm CMB}$ 導出は数十%の水準で破綻する。すなわち $k = 1$ は観測によって既に検証された帰結であり、電子質量の値（ $m_e < T_{\nu{\rm dec}} < m_\mu$ ）は本定理の整合条件が物理的に実現されるための質量階層として再解釈される。

#### Spin( $n^\ast-1$ ) 構造対応

本理論の基本方程式を構成する $\mathrm{so}(n^\ast-1) = \mathrm{so}(10)$ （§6.4 の全45チャンネル）は、大統一理論 SO(10) のゲージ代数そのものであり、(1) ゲージボソン数 $= \dim \mathrm{so}(10) = 45$ 、(2) $\mathrm{Spin}(10)$ のカイラルスピノル次元 $= 16 = 2^{n_{\rm obs}+1}$ が標準模型1世代（右巻きニュートリノ込み）を厳密に収容すること、(3) 世代数 $= n_{\rm obs} = 3$ （証明済み）、の3点が一致する。これらは確立されたGUTの事実と本理論の量の厳密な一致である。

#### 有効自由度の代数的表現

再加熱時有効自由度は次のとおりである。

$$g^\ast_{\rm reh} = (n^\ast - 1)^2 = 100$$

これは $S^2$ 上の球面調和モードの総数

$$\sum_{\ell=0}^{n^\ast-2}(2\ell+1) = (n^\ast-1)^2$$

であり、有限体 $\mathbb{F}\_{11}$ の乗法群の位数の二乗 $|\mathbb{F}_{11}^\ast|^2 = 10^2$ とも一致する。

現在のエントロピー的有効自由度は次のとおりである。

$$g^\ast_{s,{\rm today}} = 2 + \frac{7}{8}\times 6 \times \frac{4}{n^\ast} = \frac{4n^\ast - 1}{n^\ast} = \frac{43}{11}$$

分母に $n^\ast = 11$ が現れるのは、 $T_\nu/T_\gamma = (4/n^\ast)^{1/3}$ の帰結であり、偶然ではない。

#### $T_{\rm CMB}$ の完全な閉じた式

$$\boxed{T_{\rm CMB} = T_P \cdot \left[e^{-90\pi} \cdot \frac{2}{n_{\rm obs}\,\pi} \cdot \frac{30}{\pi^2} \cdot \frac{1}{(n^\ast-1)^2} \cdot \left(\frac{4n^\ast-1}{n^\ast(n^\ast-1)^2}\right)^{4/3}\right]^{1/4}}$$

この式に含まれる全ての量は $n^\ast = 11$ 、 $n_{\rm obs} = 3$ から代数的に決まる。外部パラメータは一切含まない。

数値検証：

$$T_{\rm CMB} = 2.7285\,{\rm K} \quad \text{（観測値 } 2.7255\,{\rm K}\text{、誤差 } 0.11\%\text{）}$$

各因子の証明状況は次のとおりである。

#### 補完： $g^\ast_{\rm reh}$ と標準模型の差

標準模型の高温有効自由度 $g^\ast_{\rm SM} = 106.75$ との差は次のとおりである。

$$\Delta g = g^\ast_{\rm SM} - g^\ast_{\rm reh} = 106.75 - 100 = \frac{27}{4} = \frac{n_{\rm obs}^3}{4}$$

$n_{\rm obs}^3 = 27$ は3次元空間の体積的自由度、 $4 = 2^2$ は最小スピン基底数の二乗に対応する。

$\Delta g$ はデータが裁定済みの反証可能予測である。 $T_{\rm CMB} \propto (g^\ast_{\rm reh})^{-1/4}$ であるから、もし再加熱期の有効自由度が標準模型高温値 $106.75$ なら $T_{\rm CMB} = 2.7285 \times (100/106.75)^{1/4} = 2.6843\,{\rm K}$ （誤差 $1.51\%$ ）となる。理論値 $g^\ast_{\rm reh} = (n^\ast-1)^2 = 100$ の誤差 $0.11\%$ に対し、データは $100$ を14倍の精度差で選好する。すなわち本理論は、再加熱期に熱化している自由度は $106.75$ ではなく $100$ であると予測し、この予測は現行データと整合している。

#### $\Delta g$ の同定と $T_{\rm reh}$ への変換

標準模型の有効自由度 $g^\ast(T)$ は質量を持つ4種（ $t, W, Z, H$ ）のBoltzmann抑制により高温値 $106.75$ から温度とともに減少する。質量入り熱力学積分（フェルミ・ボーズ分布の厳密な数値積分）により次が成立する。

$$g^\ast(T) = 100 \iff T = 63.6\ {\rm GeV}$$

この温度における欠損 $27/4 = 6.75$ の内訳は次のとおりである。

$$\Delta g = \underbrace{4.81}_{t} + \underbrace{0.99}_{W} + \underbrace{0.61}_{Z} + \underbrace{0.33}_{H} = 6.750$$

すなわち $27/4$ は電弱スケール $T \approx 64$ GeV における4つの重い標準模型粒子の半相対論的抑制分であり、本理論の $g^\ast_{\rm reh} = (n^\ast - 1)^2 = 100$ は再加熱温度の予測

$$T_{\rm reh} = 63.6\ {\rm GeV} \quad \text{（電弱スケール）}$$

に変換される。この同定は標準模型の質量スペクトル（経験的入力）を条件とするが、与えられた質量のもとで一意である。自由気体近似のQCD・電弱相互作用補正は $g^\ast$ を $\pm 1$ 〜 $\pm 2$ 程度シフトさせ、勾配 $dg^\ast/dT = 0.155$ /GeV による伝播で $T_{\rm reh} = 63.6 \pm (7 \text{〜} 14)\ {\rm GeV}$ となる。

質量依存の縮約定理： 本理論の全主張が標準模型の質量スペクトルに依存するのは、単一の組み合わせ $T(g^\ast = 100)$ を通じてのみである。 $m_e < T_{\nu{\rm dec}} < m_\mu$ は定理11.5.3により入力ではなく整合条件の物理的実現であり、 $\Delta g$ 同定と $T_{\rm reh}$ は質量スペクトルの関数 $T(g^\ast = 100)$ のみに依存し、他の全導出（ $\Lambda$ 、CMB、 $T_{\rm CMB}$ 、 $n_s$ 等）は質量を参照しない。したがって標準模型質量スペクトルの導出は本理論のいかなる主張にも必要でない。

移管定理： $T_{\rm reh}$ の無条件化は本理論の未解決問題ではなく、標準模型側の問題である。本理論がこのセクターで導出する量は無次元の自由度数 $g^\ast_{\rm reh} = (n^\ast-1)^2 = 100$ であり、これは無条件かつ既に $T_{\rm CMB}$ を通じて誤差 $0.11\%$ で検証済みである。温度というラベルは写像 $g^\ast_{\rm SM}(T)$ （標準模型の質量スペクトルが定める経験的関数）による像 $T(g^\ast = 100) = 63.6$ GeV として与えられ、この写像の第一原理導出は標準模型の質量起源問題と同一である。本理論側の主張（ $g^\ast_{\rm reh} = 100$ ）は完結している。

### 11.6 公理系と前提の最終整理

本理論が依拠する前提は次の3階層に整理される。

第一に、公理Ω（非退化性を含む）である。

第二に、既存前提2件（連続実装・等方性）である。いずれも基本方程式の時点で既に使用されていたものである。

第三に、全標準理論と共有される標準級前提（GR＋Boltzmann転写・テンソル積合成・チャンネル同定・Fock表現の選択）である。

理論固有の未解決公準は零である。基礎公準P0〜P4は導出により解決され、その依拠は上記3階層に帰着する。指数同定（P3）はΩ非退化性の公理規定により無条件の定理として成立し、独立性証明によりこの規定が最小十分であることが確定している（[改訂・監査記録 §3.3](08_revision_history.md#33-第三次検証による最終解決)）。

本理論が依拠する外部数学定理は Hurwitz定理・コーシー関数方程式・Wall定理・Stoneの定理・Schurの補題・等質空間定理・スペクトル定理・Minakshisundaram-Pleijel熱核展開であり、全て確立されたものである。

同定公準G（§9.11.4）は予想級として登録され、本理論の既存のいかなる導出にも使用されない。

各前提の解決経緯と監査の詳細は[改訂・監査記録](08_revision_history.md)に記す。

#### 基底状態前提の除去（[改訂・監査記録 §3.3](08_revision_history.md#33-第三次検証による最終解決) 定理11.10.3 の要点）

下流の全使用箇所（定理9.8.2 Step 5、補題B1）が必要とするのは状態の基底性ではなく、チャンネル間の期待値と分散の同一性である。状態が $SO(n)$ 不変であれば同一性が従い、かつ不変性はハミルトニアンが $SO(n)$ 不変である限り、相互作用やカスケード結合の有無によらずユニタリ発展で保存される。 $\rho_j$ はチャンネル座標の2次形式であり符号反転に不感であるから、任意の置換 $\pi$ に対し $\rho_j \mapsto \rho_{\pi(j)}$ と共変に変換し、不変状態の下では $(\delta\rho_1, \ldots, \delta\rho_n)$ の結合分布が置換不変となって全ての周辺分布が一致する。すなわち $\langle\rho_j\rangle$ と $\mathrm{Var}(\delta\rho_j)$ は $j$ に依らない。励起が生じても等方性が保たれる限り等分配は成立する。

#### 多チャンネルHilbert空間の明示構成（[改訂・監査記録 §3.3](08_revision_history.md#33-第三次検証による最終解決) の要点）

構成： チャンネルごとのHilbert空間を次のとおり積む。

$$\mathcal{H}_n = \bigotimes_{j=1}^{n} h_j, \qquad h_j = L^2(\mathbb{R})$$

ここで $h_j$ はチャンネル $j$ の正準対 $(q_j, p_j)$ を担う。 $SO(n)$ は座標 $(q_1, \ldots, q_n)$ の回転として作用する。 $SO(n)$ 不変な2次形式ハミルトニアンはSchurの補題により

$$H = a\sum_{j=1}^{n} p_j^2 + b\sum_{j=1}^{n} q_j^2 \qquad (a, b > 0)$$

に限られる。よって構成内でチャンネルは自動的に非結合であり、定理9.8.2 Step 3 の独立性は構成の定理となる。不変Gauss基底状態では交差共分散も厳密に零である。残る前提はテンソル積合成（標準量子力学の合成公準）とチャンネル同定の2件であり、いずれも標準級前提である。

## 11.7 批判への応答

物理学者から提起される3つの根本的批判への応答を示す。

第一に、基本方程式が恒等式ではないかという批判である。 $\alpha$ を基本方程式と独立に $\Lambda$ 観測値から逆算すると $\alpha_\Lambda = 117.920$ が得られ、理論値 $\alpha_\star = 117.913$ と $0.006\%$ で一致する。この二経路一致は基本方程式が独立な制約であることの証拠である。

第二に、観測値からの逆算ではないかという批判である。 $n^\ast = 11$ は Hurwitz 定理（1898）と Cayley-Dickson 構成から現代の観測値を一切使わず決まり、 $\alpha$ はそこから基本方程式で決まる。 $\Lambda$ は事後的な予測値（誤差 1.71%）であってフィッティング結果ではない。さらに、理論パラメータ群が縮重度の代数と独立な第一原理経路（Boltzmannコード直接計算）から全6ピークを誤差0.65%以内で再生することが検証され（§8.8）、 $\ell_1 = 220$ は後付けという批判への定量的応答が完成している。

第三に、要請が恣意的ではないかという批判である。全6要請（B1・B2・O1・O2・M・P）が公理Ωから演繹されることが証明され（§11.2.1）、この批判は根本的に解消された。
