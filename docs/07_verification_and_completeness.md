### 11.5 CMB温度 $T_{\rm CMB}$ の導出（v4.0）

v4.0 更新： $T_{\rm CMB}$ は本理論から代数的に導出可能であることが判明した。

#### 導出の核心： $n^\ast = 11$ の二重役割

$n^\ast = 11$ は二つの独立な文脈で同じ値として現れる。

役割1（スケール）： 再加熱エネルギー密度

$$\rho_{\rm inf} = \rho_P \cdot (n^\ast)^{-\alpha} \cdot \frac{2}{n_{\rm obs}\,\pi}$$

役割2（温度比）： ニュートリノ-光子温度分離

$$\frac{T_\nu}{T_\gamma} = \left(\frac{4}{n^\ast}\right)^{1/3} = \left(\frac{4}{11}\right)^{1/3}$$

役割2の代数的起源： $e^+e^-$ 消滅前後のエントロピー保存において

$$g^{\ast}_{s}(\gamma + e^+e^-) = 2 + \frac{7}{8}\times 4 = \frac{4+7}{2} = \frac{n^\ast}{2}$$

という分解が現れる。分子 $n^\ast = 11 = 4 + 7$ は「光子の統計的自由度（ $2 \times 2 = 4$ ）」と「フェルミオン統計重み（ $7$ ）」の和として代数的に決まり、本理論が公理Ωから独立に導出した $n^\ast = 11$ と一致する。

#### 定理11.5.1（熱力学的 $n^\ast$ の内部導出と $n_{\rm obs} = 3$ の一意性）（v4.7）

v4.0 の時点では、上式の各因子（ $2$ ・ $7/8$ ・ $4$ ）は標準模型の粒子内容（光子の偏極2・フェルミオン統計重み・電子陽電子の自由度4）からの借用であった。本定理はこれら全てを $n_{\rm obs}$ のみの関数として内部化し、さらに幾何学的 $n^\ast$ との一致が $n_{\rm obs} = 3$ でのみ成立することを証明する。

**準備（3つの確立された数学・物理定理）：**

(i) ゲージ自由度定理： $d = n_{\rm obs}+1$ 次元時空の質量ゼロベクトル場の物理自由度は $d - 2 = n_{\rm obs} - 1$ 。

(ii) スピノル次元定理： $d$ 次元時空（ $d$ 偶数）のDirac場のオンシェル自由度（粒子・反粒子×ヘリシティ）は $2^{d/2} = 2^{(n_{\rm obs}+1)/2}$ 。これはClifford代数 $\mathrm{Cl}(1, n_{\rm obs})$ の既約表現次元から従う。

(iii) 統計重み定理： $n_{\rm obs}$ 空間次元の質量ゼロ気体のフェルミ・ボーズ間のエネルギー密度比およびエントロピー密度比は、ディリクレ・イータとゼータの比

$$\frac{\eta(n_{\rm obs}+1)}{\zeta(n_{\rm obs}+1)} = 1 - 2^{-n_{\rm obs}}$$

である（ $\eta(s) = (1-2^{1-s})\zeta(s)$ ）。 $n_{\rm obs} = 3$ で $7/8$ 。

**定義：** $e^+e^-$ 消滅直前のエントロピー的有効自由度を内部量のみで

$$g^\ast_s = \underbrace{(n_{\rm obs}-1)}_{\text{(i)}} + \underbrace{\left(1 - 2^{-n_{\rm obs}}\right)}_{\text{(iii)}} \cdot \underbrace{2^{(n_{\rm obs}+1)/2}}_{\text{(ii)}}$$

と定義し、熱力学的観測限界次元数を $n^\ast_{\rm th} := 2\,g^\ast_s$ 、幾何学的観測限界次元数を $n^\ast_{\rm geom} := n_{\rm obs} + 2^{n_{\rm obs}}$ （§5）とする。

**定理：** $n^\ast_{\rm th} = n^\ast_{\rm geom}$ が成立する必要十分条件は $n_{\rm obs} = 3$ であり、そのとき共通値は $11$ である。

**証明：** $n_{\rm obs} = 3$ で $n^\ast_{\rm th} = 2[2 + \tfrac{7}{8}\cdot 4] = 11 = 3 + 8 = n^\ast_{\rm geom}$ （十分性）。一意性： $n \ge 4$ では

$$n^\ast_{\rm geom} - n^\ast_{\rm th} = 2^n - 2^{\lfloor(n+1)/2\rfloor+1}\left(1-2^{-n}\right) + n - 2(n-1)$$

であり、 $n \ge 4$ のとき $n - \lfloor(n+1)/2\rfloor - 1 \ge 1$ から $2^n \ge 2 \cdot 2^{\lfloor(n+1)/2\rfloor+1}$ となるため差は正、かつ $n$ について単調増大する（数値確認： $n=4$ で $6.5$ 、 $n=15$ で $32243$ ）。 $n = 1, 2$ は直接計算で不一致（ $2 \ne 3$ 、 $5 \ne 6$ ）。よって一致は $n_{\rm obs} = 3$ のみ。 $\square$

**系11.5.2：** スピノル自由度が四元数の実次元と一致する条件 $2^{(n_{\rm obs}+1)/2} = \dim_{\mathbb{R}}\mathbb{H} = 4$ もまた $n_{\rm obs} = 3$ でのみ成立する。すなわち $T_\nu/T_\gamma = (4/n^\ast)^{1/3}$ の分子 $4$ は、 $\mathrm{Spin}(3) \cong Sp(1) \subset \mathbb{H}$ （単位四元数群）のスピノル表現空間が $\mathbb{H}$ 自身であることの帰結として、Cayley-Dickson 塔の第2段代数 $\mathbb{H}$ に内部化される。

**帰結（借用問題の解消）：** $T_\nu/T_\gamma = (4/n^\ast)^{1/3}$ に現れる全ての数は標準模型の粒子表からの借用ではなく、(i)〜(iii) の確立された定理を経由した $n_{\rm obs}$ の関数である。さらに熱力学経路と幾何学経路（§5）が同一の $n^\ast = 11$ を与えることは仮定ではなく、 $n_{\rm obs} = 3$ を一意に選び出す整合性定理である。

#### 定理11.5.3（消滅種数 $k = 1$ の一意性：熱史同定の内部導出）（v4.11）

v4.7 の定理11.5.1 に残っていた前提「ニュートリノ脱結合から現在までに消滅する種が最小Dirac場1種である」は、実は前提ではなく定理の帰結である。

定理： 消滅する最小Dirac種の個数を $k$ とし、

$$n^\ast_{\rm th}(n_{\rm obs}, k) = 2\left[(n_{\rm obs}-1) + \left(1-2^{-n_{\rm obs}}\right) k\, 2^{(n_{\rm obs}+1)/2}\right]$$

と一般化する。 $n^\ast_{\rm th}(n_{\rm obs}, k) = n^\ast_{\rm geom}(n_{\rm obs}) = n_{\rm obs} + 2^{n_{\rm obs}}$ を満たす整数解は $(n_{\rm obs}, k) = (3, 1)$ のみである。

**証明：** $n_{\rm obs} = 3$ では条件は $4 + 7k = 11$ となり $k = 1$ が唯一の解（ $k=2$ は $18$ 、 $k=3$ は $25$ ）。格子 $1 \le n_{\rm obs} \le 15$ 、 $1 \le k \le 10$ の全数走査で解は $(3,1)$ のみであることを厳密分数計算で確認した。 $n_{\rm obs} \ge 4$ では $2^{n_{\rm obs}}$ の成長がフェルミオン項の成長 $k \cdot 2^{(n_{\rm obs}+1)/2}$ を任意の固定 $k$ で支配するため、大きい $n_{\rm obs}$ にも解は存在しない。 $\square$

**物理的必然性：** もし2種が消滅すれば $T_\nu/T_\gamma = (4/18)^{1/3}$ となり、誤差0.11%で成功している $T_{\rm CMB}$ 導出（§11.5）は数十%の水準で破綻する。すなわち $k = 1$ は観測によって既に検証された帰結であり、電子質量の値（ $m_e < T_{\nu{\rm dec}} < m_\mu$ ）は本定理の整合条件が物理的に実現されるための質量階層として再解釈される。定理11.5.1は前提なしの定理に格上げされた。

#### Spin( $n^\ast-1$ ) 構造対応（ $\Delta g$ 問題への経路）（v4.11）

$\Delta g = n_{\rm obs}^3/4$ の完全導出は標準模型の粒子内容の導出を要求するが、その経路となる厳密な構造対応が存在する。本理論の基本方程式を構成する $\mathrm{so}(n^\ast-1) = \mathrm{so}(10)$ （§6.4 の全45チャンネル）は、大統一理論 SO(10) のゲージ代数そのものであり、(1) ゲージボソン数 $= \dim \mathrm{so}(10) = 45$ 、(2) $\mathrm{Spin}(10)$ のカイラルスピノル次元 $= 16 = 2^{n_{\rm obs}+1}$ が標準模型1世代（右巻きニュートリノ込み）を厳密に収容すること、(3) 世代数 $= n_{\rm obs} = 3$ （証明済み§9.4）、の3点が一致する。これらは確立されたGUTの事実と本理論の量の厳密な一致であり、 $\Delta g$ の導出は「 $\mathrm{Spin}(10)$ 対称性の破れの連鎖を本理論から導く」という明確に定式化された条件付き課題に変換された。対応自体は事実、破れの連鎖の導出は未達である。

#### 有効自由度の代数的表現

再加熱時有効自由度：

$$g^\ast_{\rm reh} = (n^\ast - 1)^2 = 100$$

これは $S^2$ 上の球面調和モードの総数 $\sum_{\ell=0}^{n^\ast-2}(2\ell+1) = (n^\ast-1)^2$ であり、有限体 $\mathbb{F}_{11}$ の乗法群の位数の二乗 $|\mathbb{F}_{11}^\ast|^2 = 10^2$ とも一致する。

現在のエントロピー的有効自由度：

$$g^\ast_{s,{\rm today}} = 2 + \frac{7}{8}\times 6 \times \frac{4}{n^\ast} = \frac{4n^\ast - 1}{n^\ast} = \frac{43}{11}$$

分母に $n^\ast = 11$ が現れるのは、 $T_\nu/T_\gamma = (4/n^\ast)^{1/3}$ の帰結であり、偶然ではない。

#### $T_{\rm CMB}$ の完全な閉じた式

#### 補完： $g^\ast_{\rm reh}$ と標準模型の差

標準模型の高温有効自由度 $g^\ast_{\rm SM} = 106.75$ との差：

$$\Delta g = g^\ast_{\rm SM} - g^\ast_{\rm reh} = 106.75 - 100 = \frac{27}{4} = \frac{n_{\rm obs}^3}{4}$$

$n_{\rm obs}^3 = 27$ は3次元空間の体積的自由度、 $4 = 2^2$ は最小スピン基底数の二乗に対応する。この差は本理論の $g^\ast_{\rm reh} = (n^\ast-1)^2$ が「 $S^2$ 球面調和構造に収まるモード」を計上しているのに対し、標準模型では追加で $n_{\rm obs}^3/4 = 6.75$ の自由度が電弱スケール付近に存在することを示す。

再解釈（v4.12）： $\Delta g$ は説明すべき不一致ではなく、データが裁定済みの反証可能予測である。 $T_{\rm CMB} \propto (g^\ast_{\rm reh})^{-1/4}$ であるから、もし再加熱期の有効自由度が標準模型高温値 $106.75$ なら $T_{\rm CMB} = 2.7285 \times (100/106.75)^{1/4} = 2.6843\,{\rm K}$ （誤差 $1.51\%$ ）となる。理論値 $g^\ast_{\rm reh} = (n^\ast-1)^2 = 100$ の誤差 $0.11\%$ に対し、データは $100$ を14倍の精度差で選好する。すなわち本理論は「再加熱期に熱化している自由度は $106.75$ ではなく $100$ である」と予測し、この予測は現行データと整合している。残る課題は $106.75 - 100 = 27/4$ に対応する自由度（電弱スケール付近）の同定であった。

同定の完了（v4.13）： 標準模型の有効自由度 $g^\ast(T)$ は質量を持つ4種（ $t, W, Z, H$ ）のBoltzmann抑制により高温値 $106.75$ から温度とともに減少する。質量入り熱力学積分（フェルミ・ボーズ分布の厳密な数値積分）により：

$$g^\ast(T) = 100 \iff T = 63.6\ {\rm GeV}$$

であり、この温度における欠損 $27/4 = 6.75$ の内訳は

$$\Delta g = \underbrace{4.81}_{t} + \underbrace{0.99}_{W} + \underbrace{0.61}_{Z} + \underbrace{0.33}_{H} = 6.750$$

と厳密に同定される。すなわち $27/4$ は「電弱スケール $T \approx 64$ GeV における4つの重い標準模型粒子の半相対論的抑制分」であり、本理論の $g^\ast_{\rm reh} = (n^\ast - 1)^2 = 100$ は再加熱温度の予測

$$T_{\rm reh} = 63.6\ {\rm GeV} \quad \text{（電弱スケール）}$$

に変換される。この同定は標準模型の質量スペクトル（経験的入力）を条件とするが、与えられた質量のもとで一意である。

不確かさと $m_H/2$ 近接の判定（v4.14）： 上の計算は自由気体近似であり、QCD・電弱相互作用補正は $g^\ast$ を $\pm 1$ 〜 $\pm 2$ 程度シフトさせる。勾配 $dg^\ast/dT = 0.155$ /GeV による伝播で：

$$T_{\rm reh} = 63.6 \pm (7 \text{〜} 14)\ {\rm GeV}$$

v4.13 で観測事実として記録した $m_H/2 = 62.6$ GeV との近接（差 $1.0$ GeV）は、この不確かさに対して有意でなく、説明を要する事実ではないと判定する（記録を訂正）。

質量依存の縮約定理（v4.14）： 本理論の全主張が標準模型の質量スペクトルに依存するのは、単一の組み合わせ $T(g^\ast = 100)$ を通じてのみである。証明：(1) $m_e < T_{\nu{\rm dec}} < m_\mu$ は定理11.5.3により入力ではなく整合条件の物理的実現である。(2) $\Delta g$ 同定と $T_{\rm reh}$ は質量スペクトルの関数 $T(g^\ast = 100)$ のみに依存する。(3) 他の全導出（ $\Lambda$ 、CMB、 $T_{\rm CMB}$ 、 $n_s$ 等）は質量を参照しない。したがって「標準模型質量スペクトルの導出」は本理論のいかなる主張にも必要でなく、必要なのは $T_{\rm reh}$ の無条件化のみである。 $\square$

移管定理（v4.15）： $T_{\rm reh}$ の無条件化は本理論の未解決問題ではなく、標準模型側の問題である。証明：本理論がこのセクターで導出する量は無次元の自由度数 $g^\ast_{\rm reh} = (n^\ast-1)^2 = 100$ であり、これは無条件かつ既に $T_{\rm CMB}$ を通じて誤差 $0.11\%$ で検証済みである。「温度」というラベルは写像 $g^\ast_{\rm SM}(T)$ （標準模型の質量スペクトルが定める経験的関数）による像 $T(g^\ast = 100) = 63.6 \pm (7$ 〜 $14)$ GeV として与えられる。縮約定理によりこの写像は本理論のいかなる導出にも使われず、逆向きの参照（理論の出力 $100$ を温度に翻訳する）にのみ現れる。よって無条件化の課題は「写像 $g^\ast_{\rm SM}(T)$ の第一原理導出」すなわち標準模型の質量起源問題と同一であり、本理論の課題リストから標準模型の未解決問題へ移管される。本理論側の主張（ $g^\ast_{\rm reh} = 100$ ）は完結している。 $\square$

## 11.6 v4.3 批判応答：3つの根本的批判への明示的回答

v4.3 では物理学者から提起される3つの根本的批判に対して明示的な応答を追加した。

（1）「基本方程式は恒等式ではないか」： $\alpha$ を基本方程式と独立に $\Lambda$ 観測値から逆算すると $\alpha_\Lambda = 117.920$ が得られ、理論値 $\alpha_\star = 117.913$ と $0.006\%$ で一致する。この二経路一致は基本方程式が独立な制約であることの証拠である。

（2）「観測値からの逆算ではないか」： $n^\ast = 11$ は Hurwitz 定理（1898）と Cayley-Dickson 構成から現代の観測値を一切使わず決まり、 $\alpha$ はそこから基本方程式で決まる。 $\Lambda$ は事後的な予測値（誤差 1.71\%）であってフィッティング結果ではない。さらに v4.6 では、理論パラメータ群が縮重度の代数と独立な第一原理経路（Boltzmannコード直接計算）から全6ピークを誤差0.65%以内で再生することが検証され（§8.8）、「 $\ell_1 = 220$ は後付け」という批判への定量的応答が完成した。

（3）「要請が恣意的ではないか」： 要請O1は八元数の非結合性が $(A \circ B)^{-1} = B^{-1} \circ A^{-1}$ を破ることの代数的証明により正当化され、要請O2は $SO(3)$ の必要条件として精密化された。要請M・Pは Ockham の剃刀とユニタリ性という物理学の普遍的方法論的前提であり、本理論固有の恣意的選択ではない。なお v4.4 ではさらに全6要請が公理Ωから演繹されることが証明され、この批判は根本的に解消された（§11.2.1 参照）。
