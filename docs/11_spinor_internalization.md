# §14： $\mathbf{16}$ 表現対応の内製化と同定要請の明示

← [§13：一致の稀少性監査](10_coincidence_audit.md) | [README（トップ）](../README.md)

本稿は §11.5 で外部に委ねられていたスピノル側の議論を、確立数学で書ける部分と同定要請として明示すべき部分に分離し、後者に番号を与える。

## 1. 確立数学で書ける部分

Cayley-Dickson塔とClifford代数の周期性： 実Clifford代数の8周期性（Bott）はWall定理 $\mathrm{sBr}(\mathbb{R}) \cong \mathbb{Z}/8$ と同根であり、§5 が $n^\ast = 11$ の決定に用いた構造と同一の塔に属する。

スピノル表現： $\mathfrak{so}(10)$ の複素半スピノル表現は $\Delta^\pm$ 、各16次元である。これは選択ではなく表現論の事実である。

分岐則： $\mathfrak{so}(10) \supset \mathfrak{su}(5) \supset \mathfrak{su}(3) \oplus \mathfrak{su}(2) \oplus \mathfrak{u}(1)$ のもとで $\mathbf{16} = \mathbf{1} \oplus \bar{\mathbf{5}} \oplus \mathbf{10}$ であり、ハイパーチャージは可換方向の正規化を除き分岐則が一意に固定する。

可換方向の一意性： 定理6.10.3により、旗安定化群 $U(2)$ の中心の $\mathrm{Im}\,\mathbb{O}$ 上のスペクトル $\{0, \pm 1, \pm 2\}$ がレプトンパターンと厳密一致し、整数正規化で正規化が確定する。 $\bar{\mathbf{5}} \oplus \mathbf{1}$ のレプトン成分 $(L: Y = -1,\ \nu^c: Y = 0)$ および $\mathbf{10}$ の $e^c: Y = +2$ は、この割当の共役を含めて整合する。

以上により、同定要請を認めた後の構造は全て定理である。

## 2. 同定要請（番号付き）

要請R1（物質同定）： 物理的フェルミオン1世代は $\mathfrak{so}(10)$ の既約半スピノル加群 $\Delta^+$ である。

地位： 独立公理。塔の表現論は $\Delta^+$ の存在を保証するが、物質がそれであることを強制しない。

要請R2（カイラリティ同定）： カイラリティは $\sigma$ パリティの次数付けである。

地位： 独立公理。§6.8 の奇パリティ整合は支持証拠だが証明ではない。

要請R3（可換方向同定）： ハイパーチャージ方向は旗安定化群の中心である。

地位： 定理に昇格済み（定理6.10.3、一意性はレプトンパターン一致による）。

要請R4（世代指標同定）： 物質スピノル場は観測空間の虚四元数方向 $\mathrm{Im}\,\mathbb{H}$ を世代指標として担う。世代数は $N_g = \dim_{\mathbb{R}} \mathrm{Im}\,\mathbb{H}$ である。

地位： 独立公理（同定公理A10）。この公理のもとで世代数は定理として確定する。物質場の世代指標を観測空間の虚四元数構造に同定する規定は確立数学ではなく物理的同定であり、so(10)スピノル加群の表現論的重複度は1であるから、公理を欠くと世代数は決まらない。

#### 定理14.2.1（世代数の確定）

公理A10のもとで世代数は次のとおり確定する。

$$N_g = \dim_{\mathbb{R}} \mathrm{Im}\,\mathbb{H} = 3$$

**証明：** Hurwitz定理（§4.2、証明済み）により結合的除算代数で観測要請を満たすのは四元数 $\mathbb{H}$ であり、その虚部の実次元は $\dim_{\mathbb{R}} \mathrm{Im}\,\mathbb{H} = 3$ である。公理A10は物質スピノル場の世代指標をこの $\mathrm{Im}\,\mathbb{H}$ に同定するから、独立な世代の個数は $\mathrm{Im}\,\mathbb{H}$ の次元に一致する。 $\square$

地位の明示： 本定理は公理A10を前提とする条件付き定理である。so(10)の半スピノル表現は1個の既約 $\mathbf{16}$ であり、その反復重複度を3に固定する構造はso(10)単独には存在しない。本理論はこの重複度を、表現論からではなく観測空間 $\mathrm{Im}\,\mathbb{H}$ の3次元構造（Hurwitzで確定済み）への同定として与える。これは標準的なso(10)大統一が世代数を外部入力として手で与えるのとは異なり、本理論内で幾何学的に確定済みの3次元構造を世代の住処として公理化する点で固有である。Weinberg角と同様、群構造や表現論からは決まらない量を、本理論固有の確定済み構造への1個の同定公理で閉じる。

## 3. 帰結

R1・R2 を公理在庫（A9の細分）に加えれば、標準模型1世代の表現内容・電荷量子化・群構造 $SU(3) \times SU(2) \times U(1)$ は全て定理である。R4（公理A10）を加えれば世代数 $N_g = 3$ も定理14.2.1として確定する。委ねる構造は解消され、依存関係は公理として可視化された。世代数はso(10)の表現論からは決まらず、観測空間 $\mathrm{Im}\,\mathbb{H}$ への同定公理A10を経由してのみ確定する。
