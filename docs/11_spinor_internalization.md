# §14： $\mathbf{16}$ 表現対応の内製化と同定要請の明示

← [§13：一致の稀少性監査](10_coincidence_audit.md) | [README（トップ）](../README.md)

本稿は §11.5 で外部に委ねられていたスピノル側の議論を、確立数学で書ける部分と同定要請として明示すべき部分に分離し、後者に番号を与える。

## 1. 確立数学で書ける部分

Cayley-Dickson塔とClifford代数の周期性： 実Clifford代数の8周期性（Bott）はWall定理 $\mathrm{sBr}(\mathbb{R}) \cong \mathbb{Z}/8$ と同根であり、§5 が $n^\ast = 11$ の決定に用いた構造と同一の塔に属する。

スピノル表現： $\mathfrak{so}(10)$ の複素半スピノル表現は $\Delta^\pm$ 、各16次元である。これは選択ではなく表現論の事実である。

分岐則： $\mathfrak{so}(10) \supset \mathfrak{su}(5) \supset \mathfrak{su}(3) \oplus \mathfrak{su}(2) \oplus \mathfrak{u}(1)$ のもとで $\mathbf{16} = \mathbf{1} \oplus \bar{\mathbf{5}} \oplus \mathbf{10}$ であり、ハイパーチャージは可換方向の正規化を除き分岐則が一意に固定する。

可換方向の一意性： 定理6.10.3により、旗安定化群 $U(2)$ の中心の $\mathrm{Im}\,\mathbb{O}$ 上のスペクトル $\{0, \pm 1, \pm 2\}$ がレプトンパターンと厳密一致し、整数正規化で正規化が確定する。 $\bar{\mathbf{5}} \oplus \mathbf{1}$ のレプトン成分 $(L: Y = -1,\ \nu^c: Y = 0)$ および $\mathbf{10}$ の $e^c: Y = +2$ は、この割当の共役を含めて整合する。

以上により、同定要請を認めた後の構造は全て定理である。

## 2. 同定要請（番号付き）

要請R1（物質同定）： 物理的フェルミオン1世代は $\mathfrak{so}(10)$ の既約半スピノル加群 $\Delta^+$ である。

地位： 独立公理。塔の表現論は $\Delta^+$ の存在を保証するが、物質がそれであることを強制しない。

要請R2（カイラリティ同定）： カイラリティは $\sigma$ パリティの次数付けである。

地位： 独立公理。§6.8 の奇パリティ整合は支持証拠だが証明ではない。

要請R3（可換方向同定）： ハイパーチャージ方向は旗安定化群の中心である。

地位： 定理に昇格済み（定理6.10.3、一意性はレプトンパターン一致による）。

要請R4（世代数）： $N_g = 3$ 。

地位： 未導出。本理論は世代数を予測しない。定理11.5.3は消滅種数 $k = 1$ を扱うものであり世代数の導出ではない。これを導出済みと読める記述があれば訂正を要する。

## 3. 帰結

R1・R2 を公理在庫（A9の細分）に加えれば、標準模型1世代の表現内容・電荷量子化・群構造 $SU(3) \times SU(2) \times U(1)$ は全て定理である。R4 は開問題として明示的に残る。委ねる構造は解消され、依存関係は公理として可視化された。
