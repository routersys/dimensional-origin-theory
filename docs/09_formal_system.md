# §12：公理Ωの形式化と仮定の完全在庫

← [付録A〜E](appendix.md) | 次 → [§13：一致の稀少性監査](10_coincidence_audit.md)

本稿は理論の導出チェーンを形式体系として再構成し、各段が定理・定義・独立公理のいずれであるかを判定する監査文書である。目的は「仮定なし」という主張を検証可能な形に置き換えることにある。

---

## 1. 形式体系 Ω-calc の定義

### 1.1 言語

対象： 対合付き実代数 $A$ の族。初期対象は $\mathbb{R}$ 。

基本操作： 倍加 $\mathcal{D}: A \mapsto A \oplus A\,e$ （Cayley-Dickson構成）。

基本対合： $\sigma$ 。各倍加段の実切片と虚切片の組（双対対）を交換する作用として各 $\mathcal{D}(A)$ 上に誘導される。

### 1.2 公理

公理Ω0（生成）： 初期対象 $\mathbb{R}$ が存在し、 $\mathcal{D}$ は反復可能である。

公理ND（Ω非退化性）： $\mathcal{D}$ が生成する全ての双対対に対し、 $\sigma$ の誘導作用は非自明である。

この2つが §2.1 の公理Ωの形式的内容の全てである。推論規則は通常の数学（ZFC上の定理）のみを用いる。

## 3. 帰結

最小公理集合は Ω0・ND・A1〜A9 である。したがって本理論の正確な主張は次に書き換えられる。

旧主張： 全てが公理Ωのみから導出される。

新主張： 公理Ω（Ω0＋ND）と9個の同定公理から、観測と比較可能な数値行21個（ $n_{\rm obs}$ ・ $\alpha$ ・ $\Lambda$ ・CMBピーク6個・ $r$ ・ $w$ ・ $N_{\rm eff}$ ・ $n_t$ ・ $n_s$ ・ $R_{\rm dec}$ ・ $z_{\rm dec}$ ・ $H_0$ ・ $\Omega_\Lambda$ ・ $\Omega_b h^2$ ・ $T_{\rm CMB}$ ・ $\ell_A$ ）および構造的結果（ $n^\ast = 11$ ・標準模型群構造・深紫外次元 $D = 2$ ・ $T_\nu/T_\gamma$ ・次元流スケール $l(3)$ ）が導出される。21行のうち $H_0$ ・ $\Omega_\Lambda$ ・ $\ell_A$ は $\Lambda$ から従属的に決まり、ピーク $k \ge 2$ は $\ell_1$ と $\Delta\ell$ に従属するため、完全に独立な比較行は13前後である。公理1個あたりの独立出力比が理論の経済性の指標であり、これは検証可能な主張である。

旧主張の撤回は理論の弱体化ではない。同定公理を隠した「仮定なし」は一箇所の反例で全体が崩れるが、在庫を明示した形は各公理を個別に攻撃・改良・置換できる構造であり、科学的に強い。
