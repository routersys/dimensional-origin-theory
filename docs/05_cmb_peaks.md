## 8. CMB温度異方性ピーク構造の導出

### 8.1 空間球面の決定

$n^\ast$ 次元時空から時間方向を除いた空間の単位球面：

$$S^{n^\ast-2} = S^9 \quad (n^\ast = 11)$$

### 8.2 $S^9$ 上の球面調和縮重度

$S^9$（$\subset \mathbb{R}^{10}$）上の $\ell$ 次球面調和関数の縮重度（標準公式）：

$$D_\ell(S^9) = \binom{\ell+9}{\ell} - \binom{\ell+7}{\ell-2}$$

主要な値：

| $\ell$ | $D_\ell(S^9)$ | パリティ |
|--------|---------------|----------|
| 0 | 1 | 偶 |
| 1 | 10 | 奇 |
| 2 | 54 | 偶 |
| 3 | 210 | 奇 |
| 4 | 660 | 偶 |
| 5 | 1782 | 奇 |
| 6 | 4290 | 偶 |

### 8.3 奇パリティ論証によるモード選択

公理Ω（$\neg\exists\to\exists$）は奇パリティ変換である。$S^9$ 上の球面調和のパリティは：

$$Y_\ell(-\Omega) = (-1)^\ell Y_\ell(\Omega)$$

- $\ell$ = 偶数：偶パリティ（対称）→ 静的・均一な背景場 → 音響振動に寄与しない
- $\ell$ = 奇数：奇パリティ（反対称）→ 動的・音響振動モード → 観測可能

3次元観測者が観測可能な範囲（$\ell \le n_{\rm obs} = 3$）の奇パリティモード：

$$\ell \in \{1, 3\}$$

### 8.4 CMB第1ピークの代数的証明

定理：

$$D_{n_{\rm obs}}(S^{n^\ast-2}) + D_{n_{\rm obs}-2}(S^{n^\ast-2}) = \binom{n^\ast+1}{n_{\rm obs}}$$

**証明（純粋代数）：**

$n = n^\ast - 1 = 10$ として：

$$D_3(S^9) = \binom{12}{3} - \binom{10}{1} = 220 - 10 = 210$$

$$D_1(S^9) = \binom{10}{1} = 10$$

$$D_3 + D_1
= \left[\binom{12}{3} - \binom{10}{1}\right] + \binom{10}{1}
= \binom{12}{3}
= 220 \qquad \blacksquare$$

$\binom{10}{1}$ 項が完全にキャンセルする。これは $\binom{*}{-1} = 0$（二項係数の基本定理）による純粋な代数的帰結であり、外部パラメータを一切含まない。

なぜ $n_{\rm obs}=3$ でのみ成立するか：

一般形は：

$$D_{n_{\rm obs}} + D_{n_{\rm obs}-2}
= \binom{n^\ast+1}{n_{\rm obs}} - \binom{n^\ast-3}{n_{\rm obs}-4}$$

右辺第2項が消える条件は $n_{\rm obs} - 4 < 0$、すなわち $n_{\rm obs} = 3$（または1）のときのみ。Hurwitz定理が $n_{\rm obs} = 3$ を一意に決定するとき、このキャンセルが自動的に生じる。

数値的確認：

- $n_{\rm obs} = 1$：$D_1 + D_{-1} = 10 \ne 12 = \binom{12}{1}$（不成立）
- $n_{\rm obs} = 3$：$D_3 + D_1 = 220 = \binom{12}{3} = 220$（成立 ✓）
- $n_{\rm obs} = 5$：$D_5 + D_3 = 1992 \ne 792 = \binom{12}{5}$（不成立）

**結論：**

$$\boxed{\ell_1 = D_3(S^9) + D_1(S^9) = \binom{12}{3} = 220}$$

観測値 220 と誤差 0.000% で一致する。

### 8.5 組み合わせ論的意味

$\binom{n^\ast+1}{n_{\rm obs}} = \binom{12}{3} = 220$ は「$n^\ast+1 = 12$ 次元時空の中で $n_{\rm obs} = 3$ 次元断面を選ぶ独立な配置数」を表す。これが一次元起源の空間が3次元観測者に「見える」最大モード数＝第1ピーク位置となる。

### 8.6 全ピーク式（完全代数的導出）

> 重要な改訂（v2.0）： 従来の $\ell_k = 303(k-1/4)$ を、完全に代数的に導出される新式に置き換える。

$$\boxed{\ell_k = \ell_1 + (k-1) \cdot \Delta\ell}$$

$$= \binom{n^\ast+1}{n_{\rm obs}} + (k-1) \cdot \frac{D_{n_{\rm obs}+1}(S^{n^\ast-2}) - D_{n_{\rm obs}-1}(S^{n^\ast-2})}{2}$$

$$= 220 + (k-1) \times 303$$

#### 音響スケール $\Delta\ell$ の代数的導出

音響振動の繰り返し間隔（音響スケール）は、$S^9$ 上の観測者近傍（$\ell = n_{\rm obs} \pm 1$）の偶パリティモード縮重度の差から定義される：

$$\Delta\ell = \frac{D_{n_{\rm obs}+1}(S^{n^\ast-2}) - D_{n_{\rm obs}-1}(S^{n^\ast-2})}{2}$$

代数的計算（$n_{\rm obs}=3$, $n^\ast=11$）：

$$D_4(S^9) = \binom{13}{4} - \binom{11}{2} = 715 - 55 = 660$$

$$D_2(S^9) = \binom{11}{2} - \binom{9}{0} = 55 - 1 = 54$$

$$\Delta\ell = \frac{660 - 54}{2} = \frac{606}{2} = 303$$

係数 $1/2$ の起源： $D_4 - D_2$ は $S^9$ 上の進行波の両方向（正・負波数）を勘定したもの。定在波（音響モード）は2方向の進行波の重ね合わせであるから、実質的な間隔は半分となる。

#### 物理的意味

- 第1ピーク $\ell_1 = 220$： 「$S^9$ の奇パリティ最低次モードが3次元空間に見える最大縮重度」
- 音響スケール $\Delta\ell = 303$： 「観測者近傍の $S^9$ モード密度変化の半値」

全ピーク式 $\ell_k = 220 + (k-1) \times 303$ には外部パラメータが皆無であり、$n_{\rm obs}$ と $n^\ast$ のみから完全に決定される。

### 8.8 残余誤差の系統的解釈（v4.4 補強）

本理論式は純粋光子流体（バリオン負荷 $R = 0$）の極限における予測式である。本理論が導出するバリオン-光子比 $R_{\rm dec} = 3/5$（[§9.5](06_predictions.md#95-バリオン-光子比と脱結合赤方偏移)、観測値との誤差0.5%）によるピーク移動が系統誤差として存在する。

標準宇宙論の音響ピーク位相シフトの近似：

$$\delta\ell_k \approx \pm\frac{R_{\rm dec}}{1+R_{\rm dec}} \cdot \frac{\Delta\ell}{2} \cdot (-1)^k$$

$R_{\rm dec} = 3/5$ を代入すると $R/(1+R) = 3/8$、最大補正量 $\approx \pm 57$。この補正を含む精密化式は本理論の代数的課題として残る（$R_{\rm dec}$ 自体は代数的に導出済みであるため、補正式の代数的導出は原理的に可能）。

観測誤差パターンの符号（$k=2$ が正方向、$k=3$ が負方向にずれる）はバリオン負荷の偶奇非対称効果と定性的に整合する。
