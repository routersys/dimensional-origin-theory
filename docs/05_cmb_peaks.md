## 8. CMB温度異方性ピーク構造の導出

### 8.1 空間球面の決定

$n^\ast$ 次元時空から時間方向を除いた空間の単位球面：

$$S^{n^\ast-2} = S^9 \quad (n^\ast = 11)$$

### 8.2 $S^9$ 上の球面調和縮重度

$S^9$（$\subset \mathbb{R}^{10}$）上の $\ell$ 次球面調和関数の縮重度（標準公式）：

$$D_\ell(S^9) = \binom{\ell+9}{\ell} - \binom{\ell+7}{\ell-2}$$

主要な値：

| $\ell$ | $D_\ell(S^9)$ | パリティ |
|--------|---------------|----------|
| 0 | 1 | 偶 |
| 1 | 10 | 奇 |
| 2 | 54 | 偶 |
| 3 | 210 | 奇 |
| 4 | 660 | 偶 |
| 5 | 1782 | 奇 |
| 6 | 4290 | 偶 |

### 8.3 奇パリティ論証によるモード選択

公理Ω（$\neg\exists\to\exists$）は奇パリティ変換である。$S^9$ 上の球面調和のパリティは：

$$Y_\ell(-\Omega) = (-1)^\ell Y_\ell(\Omega)$$

- $\ell$ = 偶数：偶パリティ（対称）→ 静的・均一な背景場 → 音響振動に寄与しない
- $\ell$ = 奇数：奇パリティ（反対称）→ 動的・音響振動モード → 観測可能

3次元観測者が観測可能な範囲（$\ell \le n_{\rm obs} = 3$）の奇パリティモード：

$$\ell \in \{1, 3\}$$

### 8.4 CMB第1ピークの代数的証明

定理：

$$D_{n_{\rm obs}}(S^{n^\ast-2}) + D_{n_{\rm obs}-2}(S^{n^\ast-2}) = \binom{n^\ast+1}{n_{\rm obs}}$$

**証明（純粋代数）：**

$n = n^\ast - 1 = 10$ として：

$$D_3(S^9) = \binom{12}{3} - \binom{10}{1} = 220 - 10 = 210$$

$$D_1(S^9) = \binom{10}{1} = 10$$

$$D_3 + D_1
= \left[\binom{12}{3} - \binom{10}{1}\right] + \binom{10}{1}
= \binom{12}{3}
= 220 \qquad \blacksquare$$

$\binom{10}{1}$ 項が完全にキャンセルする。これは $\binom{*}{-1} = 0$（二項係数の基本定理）による純粋な代数的帰結であり、外部パラメータを一切含まない。

なぜ $n_{\rm obs}=3$ でのみ成立するか：

一般形は：

$$D_{n_{\rm obs}} + D_{n_{\rm obs}-2}
= \binom{n^\ast+1}{n_{\rm obs}} - \binom{n^\ast-3}{n_{\rm obs}-4}$$

右辺第2項が消える条件は $n_{\rm obs} - 4 < 0$、すなわち $n_{\rm obs} = 3$（または1）のときのみ。Hurwitz定理が $n_{\rm obs} = 3$ を一意に決定するとき、このキャンセルが自動的に生じる。

数値的確認：

- $n_{\rm obs} = 1$：$D_1 + D_{-1} = 10 \ne 12 = \binom{12}{1}$（不成立）
- $n_{\rm obs} = 3$：$D_3 + D_1 = 220 = \binom{12}{3} = 220$（成立 ✓）
- $n_{\rm obs} = 5$：$D_5 + D_3 = 1992 \ne 792 = \binom{12}{5}$（不成立）

**結論：**

$$\boxed{\ell_1 = D_3(S^9) + D_1(S^9) = \binom{12}{3} = 220}$$

観測値 220 と誤差 0.000% で一致する。

### 8.5 組み合わせ論的意味

$\binom{n^\ast+1}{n_{\rm obs}} = \binom{12}{3} = 220$ は「$n^\ast+1 = 12$ 次元時空の中で $n_{\rm obs} = 3$ 次元断面を選ぶ独立な配置数」を表す。これが一次元起源の空間が3次元観測者に「見える」最大モード数＝第1ピーク位置となる。

### 8.6 全ピーク式（完全代数的導出）

> 重要な改訂（v2.0）： 従来の $\ell_k = 303(k-1/4)$ を、完全に代数的に導出される新式に置き換える。
>
> 位置づけの改訂（v4.5）： 本式は位相シフト一定近似（$\varphi_k = \bar\varphi$、§8.8）における式である。$k \ge 2$ の精密予測は §8.8 の力学経路が与える。

$$\boxed{\ell_k = \ell_1 + (k-1) \cdot \Delta\ell}$$

$$= \binom{n^\ast+1}{n_{\rm obs}} + (k-1) \cdot \frac{D_{n_{\rm obs}+1}(S^{n^\ast-2}) - D_{n_{\rm obs}-1}(S^{n^\ast-2})}{2}$$

$$= 220 + (k-1) \times 303$$

#### 音響スケール $\Delta\ell$ の代数的導出

音響振動の繰り返し間隔（音響スケール）は、$S^9$ 上の観測者近傍（$\ell = n_{\rm obs} \pm 1$）の偶パリティモード縮重度の差から定義される：

$$\Delta\ell = \frac{D_{n_{\rm obs}+1}(S^{n^\ast-2}) - D_{n_{\rm obs}-1}(S^{n^\ast-2})}{2}$$

代数的計算（$n_{\rm obs}=3$, $n^\ast=11$）：

$$D_4(S^9) = \binom{13}{4} - \binom{11}{2} = 715 - 55 = 660$$

$$D_2(S^9) = \binom{11}{2} - \binom{9}{0} = 55 - 1 = 54$$

$$\Delta\ell = \frac{660 - 54}{2} = \frac{606}{2} = 303$$

係数 $1/2$ の起源： $D_4 - D_2$ は $S^9$ 上の進行波の両方向（正・負波数）を勘定したもの。定在波（音響モード）は2方向の進行波の重ね合わせであるから、実質的な間隔は半分となる。

#### 物理的意味

- 第1ピーク $\ell_1 = 220$： 「$S^9$ の奇パリティ最低次モードが3次元空間に見える最大縮重度」
- 音響スケール $\Delta\ell = 303$： 「観測者近傍の $S^9$ モード密度変化の半値」

全ピーク式 $\ell_k = 220 + (k-1) \times 303$ には外部パラメータが皆無であり、$n_{\rm obs}$ と $n^\ast$ のみから完全に決定される。

### 8.8 力学経路検証と系統誤差の解明（v4.5）

> 改訂（v4.5）： 旧版で提示したバリオン負荷位相シフト近似 $\delta\ell_k \approx \pm\frac{R_{\rm dec}}{1+R_{\rm dec}}\cdot\frac{\Delta\ell}{2}(-1)^k$（最大補正 ±57）は、数値検証の結果、観測残差（$k=2$ で +14 程度）と大きさが一致せず誤りであった。正しい機構は以下の力学経路検証により確定した。

#### 8.8.1 検証の構成

代数経路（§8.4〜§8.6）とは完全に独立な第二の経路として、本理論が導出した宇宙論パラメータ群

$$H_0 = 67.94,\quad \Omega_\Lambda = 0.690,\quad T_{\rm CMB} = 2.7285\,{\rm K},\quad \Omega_b h^2 = 0.02156,\quad z_{\rm dec} = 1089,\quad R_{\rm dec} = \tfrac{3}{5},\quad \Omega_\phi^{ls} = 0$$

（$\Omega_\phi^{ls} = 0$ は $w = -1$ の証明済み帰結、[§9.3](06_predictions.md#93-暗黒エネルギー状態方程式)）を標準音響物理に入力し、ピーク位置を力学的に計算する。観測値は一切入力しない。

音響角度スケールは平坦宇宙の積分

$$\ell_A = \pi\,\frac{D_A(z_{\rm dec})}{r_s(z_{\rm dec})},\qquad
D_A = \frac{c}{H_0}\int_0^{z_{\rm dec}}\frac{dz}{E(z)},\qquad
r_s = \frac{c}{H_0}\int_{z_{\rm dec}}^{\infty}\frac{c_s(z)}{c}\frac{dz}{E(z)}$$

で計算され（$c_s = c/\sqrt{3(1+R(z))}$、$R(z) = R_{\rm dec}(1+z_{\rm dec})/(1+z)$）、結果は：

$$\ell_A = 300.18$$

Planck観測値 $\ell_A = 301.76 \pm 0.10$ との差は 0.5%。

ピーク位置はパラメータ化 $\ell_k = \ell_A(k - \bar\varphi - \delta\varphi_k)$（Hu et al. 2001）に従い、位相シフトは Doran & Lilley（2002, astro-ph/0104486）の解析的フィット式

$$\bar\varphi = (1.466 - 0.466\,n_s)\left[a_1 r_\star^{a_2} + 0.291\,\Omega_\phi^{ls}\right],\qquad r_\star = \frac{\rho_r(z_{\rm dec})}{\rho_m(z_{\rm dec})}$$

および $\delta\varphi_2,\,\delta\varphi_3$ の各式で評価する。これらはBoltzmannコード（CMBFAST）の数値解のフィット（精度 $\Delta\bar\varphi = 0.0031$、$\Delta\delta\varphi_2 = 0.0044$、$\Delta\delta\varphi_3 = 0.0052$、$\ell$ 換算 ±1〜2）であり、観測データへのフィットではない。$r_\star$ は理論パラメータから $r_\star = 0.318$ と計算される。

#### 8.8.2 検証結果

$n_s = 0.965$ における位相シフトの理論計算値：

$$\bar\varphi = 0.268,\qquad \delta\varphi_2 = -0.046,\qquad \delta\varphi_3 = +0.035$$

| $k$ | 力学経路 | 代数式 $220+(k-1)\times303$ | 観測値（Planck） | 力学vs観測 | 代数vs観測 |
|-----|---------|------------------------------|-----------------|-----------|-----------|
| 1 | 219.8 | 220 | 220.0 | 0.08% | 0.00% |
| 2 | 533.8 | 523 | 537.5 | 0.68% | 2.70% |
| 3 | 809.8 | 826 | 810.8 | 0.13% | 1.87% |

$n_s = 1$ でも各値の変動は0.5%以内で結論は不変。力学経路と観測の残差は全てフィット式自体の近似精度の範囲内である。

#### 8.8.3 帰結

**帰結1（系統誤差の正体）：** 代数式の $k \ge 2$ 誤差（2.7%、1.9%）の正体は、$k$ 依存の相対位相シフト $\delta\varphi_k$ の無視である。$\delta\varphi_2 < 0$ は第2ピークを押し上げ、$\delta\varphi_3 > 0$ は第3ピークを押し下げる。これは観測の誤差パターン（$k=2$ で代数式が不足、$k=3$ で過剰）の符号と大きさの両方を定量的に説明する。物理的起源は重力ポテンシャル減衰による輻射駆動効果と振動流体のDoppler効果（標準音響物理）である。

**帰結2（独立性の証明）：** 力学経路は縮重度の代数を一切使用せず、第1〜3ピーク全てを1%未満で再生する。「$\ell_1 = 220$ は観測値に合わせた後付け」という批判に対し、同一のパラメータ群が独立な力学経路から $\ell_2,\,\ell_3$ まで再生することは、パラメータ群全体の整合性の直接的証拠である。

**帰結3（代数式の位置づけ）：** $\ell_k = 220 + (k-1)\times 303$ は位相シフト一定近似（$\varphi_k = \bar\varphi$）における式と位置づけられる。$\Delta\ell = 303$ と力学値 $\ell_A = 300.2$ の差0.9%はこの近似の範囲を定量化する。精密予測は力学チェーン $\Lambda \to H_0,\,\Omega_\Lambda$ と $R_{\rm dec},\,z_{\rm dec},\,T_{\rm CMB} \to \ell_A,\,\varphi_k \to \ell_k$ が担い、理論の実効的ピーク予測精度は平均1.13%から約0.3%へ改善される。

#### 8.8.4 残る留保

1. $n_s$ は本理論に予測が存在しない唯一の外部入力である（ただし結果への感度は0.5%以内）。
2. 位相シフトのフィット式の使用は「標準音響物理を本理論が内包する」前提に立つ。位相シフトの第一原理計算は標準宇宙論においても数値計算であり、本理論固有の弱点ではない。
3. $\delta\varphi_k$ の $n^\ast,\,n_{\rm obs}$ による代数的表現は未発見の課題として残る。
